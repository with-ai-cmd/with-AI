# Web集客プロジェクト — 秘書AI管轄

## ミッション
**with-ai.jp からの問い合わせを月50件に増やす。**
秘書AIがプロジェクトオーナーとして、CMO（企画・分析）とCTO（実装・デプロイ）に指示を出し、進捗を管理する。

```

  │   ├── HP改修の企画・効果検証

```

## 秘書の管理アクション

### 週次（毎週月曜・おはようルーティン内で実行）
```
秘書がやること:
1. CMOに「今週のWeb集客レポート出して」と指示
   → CMOがGA4/SCデータを集計し、以下を報告:
     - 先週のPV / 問い合わせ数 / 検索順位変動
     - 今週公開予定の記事テーマ
     - SNS投稿状況

2. KPIの進捗を確認
   → 目標に対してオントラック / 遅延を判定

3. 遅延がある場合の対応指示:
   - PV不足 → CMO: 「記事ペース上げて」or「SNS強化して」
   - CVR低い → CMO: 「CRO分析して改修案出して」→ CTO: 「実装して」
   - 速度遅い → CTO: 「PageSpeed改善して」
   - 順位下がった → CMO: 「原因分析して対策出して」

4. CEOへの週次サマリーをNotionに保存
```

### 月次（毎月1日）
```
秘書がやること:
1. CMOに月次レポート指示
   → 月間KPI全数値の集計 + 前月比 + 目標差異

2. CTOに技術SEO月次監査指示
   → 全ページの技術チェック + 課題リスト

3. 月次レビューミーティング相当の判断:
   - 来月のコンテンツ計画をCMOに指示
   - 技術改善の優先順位をCTOに指示
   - HP改修が必要なら改修企画をCMOに指示→CTOに実装指示
   - KPIの目標値調整（必要であれば）

4. CEOへの月次レポートをNotionに保存
```

### 四半期
```
秘書がやること:
1. 全KPIの達成率レビュー
2. 戦略の見直し（攻めるキーワード、注力ページの変更等）
3. 次四半期のマイルストーン設定
4. CMO/CTOへの四半期目標の再設定
```

## 秘書のエスカレーション基準

以下の場合はCEOに報告・相談する:
- 月間問い合わせが目標の50%未満
- 3ヶ月連続でPVが横ばい or 減少
- 重大な技術トラブル（サイトダウン等）
- 予算が必要な施策（広告出稿、ツール導入等）
- CMO/CTO間で方針が対立した場合
