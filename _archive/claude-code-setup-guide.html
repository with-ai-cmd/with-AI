Claude Code 導入ガイド

管理者権限不要・PowerShell不要。
3ステップで完了します。

0

事前準備（まだの人だけ）

以下2つがインストール済みか確認してください。

  • Node.js（v18以上） — nodejs.org から LTS をインストール
  • Git for Windows — gitforwindows.org からインストール
確認方法: ターミナルで node -v と打って v18以上が表示されればOK
1

Cursor のターミナルを Git Bash に切り替え

Cursor を開き、上部メニューから：

  • Terminal → New Terminal を開く
  • ターミナル右上の 「∨」ドロップダウン → Git Bash を選択
ターミナル右上の「+」の横にある「∨」をクリック → 「Git Bash」を選ぶ
cmd や PowerShell のままだと失敗する場合があります。
必ず Git Bash に切り替えてください。
2

このコマンドを実行するだけ

Git Bash のターミナルに以下をコピペして Enter：

npx -y @anthropic-ai/claude-code
ポイント: -y を付けることで確認プロンプトをスキップ。インストール不要で即実行されます。
3

認証してスタート

初回実行時にブラウザが開き、Anthropic アカウントでログインを求められます。

  • ブラウザでログイン / サインアップ
  • 認証完了後、ターミナルに戻る
  • Claude Code が使えるようになる
次回以降: 同じコマンド npx -y @anthropic-ai/claude-code で即起動できます。
?

うまくいかない場合

以下を順番に確認してください。

Node.js が認識されない

node -v

→ 何も出ない場合は Node.js を再インストールし、Cursor を再起動

npm のキャッシュをクリア

npm cache clean --force

ネットワーク確認

npm ping

→ VPN やプロキシを使っている場合はオフにして再試行
