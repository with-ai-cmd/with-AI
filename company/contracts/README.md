# with-AI株式会社 契約書テンプレート

## 契約書一覧

| 契約書 | 用途 | 甲 | 乙 |
|--------|------|----|----|
| AIKOMON業務委託契約書 | 社外AI責任者サービスの提供 | with-AI（提供者） | 顧客企業 |
| 営業代行業務委託契約書 | 営業チームへの営業代行委託 | with-AI（委託者） | 営業パートナー |
| システム開発契約書 | システム開発の受託 | with-AI（受託者） | 発注企業 |
| 秘密保持契約書（NDA） | 取引検討時の秘密情報保護 | with-AI | 相手方 |

## フォルダ構成

```
contracts/
├── README.md              ← このファイル
├── templates/             ← 印刷・送付用のdocxテンプレート
├── source/                ← 原本（md）＋一括生成スクリプト
│   ├── *.md               ← 各契約書の原本
│   └── generate_all.py    ← md→docx一括変換
└── signed/                ← 締結済み契約書の保管用
```

## 使い方

### テンプレートをそのまま使う
`templates/` フォルダのdocxを開いて空欄を埋めて使用。

### 契約書の内容を修正する
1. `source/` フォルダの該当mdファイルを修正
2. `python3 source/generate_all.py` を実行してdocxを再生成

### 共通設定
- 秘密保持の存続期間：全契約書で **5年間**
- 合意管轄：**東京地方裁判所**
