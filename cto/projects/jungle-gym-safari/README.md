# Jungle Gym Safari

## プロジェクト概要

株式会社JungleGYM から受託した、サブスクリプション型オンラインコミュニティプラットフォームの開発。
FANTS（ファンツ）を参考にしつつ、独自のEC・ライブ配信・会員特典機能を持つ Web + モバイルアプリを構築する。

## クライアント

- **会社名**: 株式会社JungleGYM
- **デザインコンセプト**: ロンハーマン / 西海岸テイスト
