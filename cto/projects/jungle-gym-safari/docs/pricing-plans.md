## コンテンツカテゴリ（5カテゴリ）

1. **美容** — スキンケア、メイク、ヘアケア
2. **自宅トレーニング** — ワークアウト動画、食事管理
3. **金融** — 投資、資産運用、マネーリテラシー
4. **健康** — メンタルヘルス、栄養学、ウェルネス
5. **スタートアップ** — 起業、ビジネス戦略、マーケティング

## 課金フロー

```
ユーザー登録
    │
    ├── プラン選択（Bronze / Silver / Gold）
    │       │
    │       ├── 決済手段選択
    │       │   ├── クレジットカード（Stripe）
    │       │   ├── PayPay（PayPay API）
    │       │   └── 口座引き落とし（GMOペイメント等）
    │       │
    │       └── サブスク開始（毎月自動課金）
    │
    └── 追加コンテンツ購入（Bronze / Silver ユーザー）
            │
            ├── コンテンツ単位で個別課金
            └── 決済手段は登録済みのものを利用
```

## 追加課金モデル

- Bronze/Silver ユーザーが無料枠を超えてコンテンツにアクセスする場合
- コンテンツカテゴリ単位で購入（月額 or 買い切り — 要検討）
- EC商品の購入（全プラン共通、プランによる割引率の差）
- イベントチケットの購入

## 決済手段の優先度

| 手段 | 優先度 | 理由 |
|---|---|---|
| クレジットカード | 必須 | サブスクの標準。Stripe で実装 |
| PayPay | 必須 | 国内ユーザーの利便性。若年層に強い |
| 口座引き落とし | 要検討 | 実装コスト高。ニーズ次第で Phase 2 |
