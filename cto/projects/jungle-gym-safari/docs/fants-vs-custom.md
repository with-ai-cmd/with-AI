# なぜ「自社プラットフォーム」なのか？

**株式会社JungleGYM 様向け**
**作成: with-AI株式会社**

## Jungle Gym Safari が目指す世界

ロンハーマンの店舗に足を踏み入れた瞬間、空気が変わる。
潮風と陽の光を感じるような空間。洗練されているのに、肩の力は抜けている。

**Jungle Gym Safari は、その空気をデジタルに再現する場所です。**

美容、トレーニング、金融、健康、ビジネス —
人生を豊かにするコンテンツが集まり、同じ感性を持つ仲間と繋がる。
それは既製品のプラットフォームでは決して作れない体験です。

## 既存プラットフォーム（FANTS等）の限界

FANTSは優れたサービスです。手軽に始められます。
しかし、**「手軽さ」と引き換えに失うもの**があります。

### 1. 売上の10〜30%が、毎月消えていく

FANTSの手数料は**売上の10〜30%**。

月商100万円なら、毎月10〜30万円。年間で120〜360万円。
月商500万円になれば、毎月50〜150万円。**年間で600〜1,800万円。**

会員が増えて、コンテンツが充実して、売上が伸びるほど、
**成功すればするほど、出ていくお金が増える仕組み**です。

### 2. 見た目が「FANTS」になる

FANTSはテンプレートベース。
どのコミュニティも、同じフレームの中に収まっている。

**Jungle Gym Safariの世界観 — 西海岸の風、ロンハーマンの空気感。**
それをテンプレートの枠の中で表現するのには限界があります。

「おしゃれなコミュニティ」ではなく、**「Jungle Gym Safariという場所」**を作りたいなら、
器から自分たちで作る必要があります。

### 3. できることに天井がある

- ライブ配信は月5時間まで（超過は別途料金）
- ECはFANTS内の簡易機能のみ
- 会員証でドリンクサービス？FANTSにはない
- 不動産仲介との連携？FANTSにはない
- YouTubeとの統合？FANTSにはない

**やりたいことが増えるたびに、「FANTSにその機能はありません」にぶつかる。**

### 4. データとお客様は、FANTSの中にいる

会員データ、視聴履歴、購買行動 —
これらはFANTSのプラットフォーム上に存在します。

FANTSが値上げしたら？方針を変えたら？最悪、サービスが終了したら？

**自分のビジネスの土台を、他社のサービスの上に建てるリスク。**

## 自社プラットフォームなら、こうなる

### 「Jungle Gym Safari」という世界をまるごと作れる

アプリを開いた瞬間から、西海岸の世界。
サンドベージュの背景、オーシャンブルーのアクセント。
フォント、アイコン、ボタンの角丸ひとつまで、
**すべてがJungle Gym Safariのためにデザインされている。**

FANTSを使っている競合は、全員が同じ「箱」の中。
Jungle Gym Safariだけが、**唯一無二の場所**になる。

### やりたいことが、全部できる

| やりたいこと | FANTS | Jungle Gym Safari |
|---|---|---|
| YouTube Liveでライブ配信（時間無制限） | 月5時間まで | 無制限 |
| Shopify連携の本格EC | 簡易EC | Shopifyの全機能 |
| 会員証を見せてドリンクサービス | 不可 | QRコード会員証 |
| 不動産仲介の特典 | 不可 | 自由に設計可能 |
| プラン別のコンテンツ細かい出し分け | 限定的 | 完全カスタム |
| 有料チケット制のライブ配信 | 不可 | Shopifyチケット + YouTube限定公開 |
| Instagram投稿の自動取り込み | 不可 | Graph API連携 |

### 売上が伸びても、コストはほぼ変わらない

```
■ FANTSの場合（売上に比例してコストが増える）

  月商 100万 → 手数料  10〜30万/月 → 年間  120〜360万円
  月商 300万 → 手数料  30〜90万/月 → 年間  360〜1,080万円
  月商 500万 → 手数料 50〜150万/月 → 年間  600〜1,800万円
                                      ↑ 成功するほど痛い

■ 自社開発の場合（ほぼ固定コスト）

  月商 100万 → コスト 約5万/月 → 年間 約60万円
  月商 300万 → コスト 約12万/月 → 年間 約150万円
  月商 500万 → コスト 約20万/月 → 年間 約240万円
                                   ↑ 売上が増えてもほぼ変わらない
```

**月商300万円の場合、年間で200〜900万円の差。**
この差が、毎年、積み上がっていきます。

### データは全て自分のもの

- 誰が、どのコンテンツを、何分見ているか
- どのプランの会員が一番アクティブか
- どの特典が一番使われているか
- いつ、なぜ解約が起きるか

**このデータがあれば、サービスは進化し続けられる。**
FANTSでは、この深さのデータは手に入りません。

### ビジネスの土台が、自分の手の中にある

プラットフォームの都合で、ある日突然ルールが変わることはない。
値上げされることも、サービスが終了することもない。

**会員データ、コンテンツ、購買履歴、全てが自社資産。**
Jungle Gym Safariは、JungleGYMのものです。

## 「でも、初期費用が高いのでは？」

はい。ゼロではありません。

ただし、今回の設計は**徹底的にコストを削っています**:

- ライブ配信 → **YouTube Live**（自前構築せず無料）
- EC → **Shopify**（自前構築せず月額約6,000円）
- 決済 → **Shopify Payments + Stripe**（既存サービスの組み合わせ）
- インフラ → **Vercel + Supabase + Railway**（月額約1.5万円）

**「作れるものは使う。作るべきものだけ作る。」**

自前で作るのは、Jungle Gym Safariの「核」の部分だけ:
- 会員管理（プラン制御の心臓部）
- コンテンツ配信（プラン別アクセス制御）
- タイムライン（コミュニティの顔）
- 会員特典（QRコード会員証、利用管理）
- 管理画面（全体を統合するダッシュボード）

**最小限の開発で、最大限の価値を。**
