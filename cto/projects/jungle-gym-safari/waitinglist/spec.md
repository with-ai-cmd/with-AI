# Jungle Gym Safari — ウェイティングリスト仕様

## 概要

サービスローンチ前にリードを獲得するためのウェイティングリストLP。
2026年6月〜8月の期間で運用し、9月1日のサービス開始時にまとめて案内を送信する。

## スコープ

ローンチ前に最速で公開できるよう、シンプルな1枚もののLPとする。

## 技術実装

| 項目 | 技術 | 備考 |
|---|---|---|
| フロントエンド | Next.js (Static Export) or Astro | 超軽量LP |
| ホスティング | Vercel or Cloudflare Pages | 無料〜低コスト |
| フォーム送信 | Supabase or Google Forms API | メール収集 |
| メール自動返信 | SendGrid or Resend | 登録完了メール |
| ドメイン | junglegym-safari.com（仮） | 要クライアント確認 |

## リード管理

- 登録データ: 名前、メールアドレス、興味カテゴリ、登録日時
- エクスポート: CSV形式で運営に共有可能
- ローンチ時: 一斉メールで正式オープン通知 + プラン選択への誘導
