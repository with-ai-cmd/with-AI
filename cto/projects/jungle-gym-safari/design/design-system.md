# Jungle Gym Safari — デザインシステム

## コンセプト

**ロンハーマン / 西海岸 / California Lifestyle**

リラックス感がありつつも洗練された、ライフスタイルブランドのような雰囲気。
高級感はあるが敷居は高くない。自然光、海、緑を感じるクリーンなデザイン。

## カラーパレット（案）

```
Primary:
  Sand         #F5F0E8    背景・ベース
  Ocean Blue   #4A90A4    アクセント・CTA
  Deep Navy    #2C3E50    テキスト・ヘッダー

Secondary:
  Sunset Coral #E8735A    ハイライト・通知
  Palm Green   #7BAE7F    成功・アクティブ
  Driftwood    #C4A882    ボーダー・セカンダリ

Neutral:
  White Sand   #FDFCFA    カード背景
  Warm Gray    #9B9B9B    サブテキスト
  Charcoal     #333333    本文テキスト
```

## タイポグラフィ（案）

| 用途 | フォント | 備考 |
|---|---|---|
| 見出し | Josefin Sans / Montserrat | 西海岸の開放感、モダン |
| 本文（英語） | Inter / DM Sans | 可読性重視 |
| 本文（日本語） | Noto Sans JP | クリーンで読みやすい |
| アクセント | Playfair Display | 高級感を出す箇所に限定使用 |

## UIコンポーネントの方向性

### カード
- 角丸（12-16px）
- 薄いシャドウ
- ホワイトサンド背景
- ゆったりしたパディング

### ボタン
- プライマリ: Ocean Blue、角丸フル（pill shape）
- セカンダリ: アウトライン、Driftwood ボーダー
- テキストボタン: アンダーライン付き

### ナビゲーション
- ボトムタブ（モバイル）: アイコン + ラベル、5タブ以内
- サイドバー（Web）: 落ち着いたSand背景

### 画像・メディア
- アスペクト比統一（16:9 メイン、1:1 サムネイル）
- 角丸適用
- ライブ配信: 赤いLIVEバッジ
