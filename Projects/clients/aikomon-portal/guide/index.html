操作ガイド

Claude Code のコマンド集・キーボード操作・トラブルシューティング

キーボード操作

Enter

入力した指示を送信します。

Esc

Claudeの回答を途中で止めます。間違った指示を出したとき、長すぎる回答を止めたいときに使います。

↑（上矢印キー）

過去に入力した指示を呼び出します。同じ指示をもう一度実行したいときに便利です。

Ctrl + C

Claude Codeを終了します。もう一度 claude と入力すれば再開できます。

Cmd + `（Mac）/ Ctrl + `（Windows）

Cursorのターミナルを開く / 閉じる。 ` キーはキーボード左上、数字の1の左隣にあります。

スラッシュコマンド

/help

使えるコマンドの一覧を表示します。困ったらまずこれ。

/help

/compact

長い会話を要約して軽くします。動作が遅くなったときに実行してください。

/compact

/status

ログイン状態とプランを確認します。認証がうまくいかないときに。

/status

/login

再ログインします。認証が切れたときに使います。

/login

/clear

会話をリセットします。最初からやり直したいときに。

/clear
困ったときの対処法

Step 1: Claude Codeを再起動する

Ctrl+Cで終了 → claude で再起動。ほとんどの問題はこれで直ります。

Step 2: 会話をリセットする

/clear と入力。会話履歴が消えて最初からやり直せます。

/clear

Step 3: ヘルスチェックする

claude doctor と入力。設定やインストールの問題を自動診断します。

claude doctor

Step 4: Cursorごと再起動する

Cursorを完全に閉じて → もう一度起動 → ターミナルを開く → claude。環境変数やPATHの問題はこれで直ります。

CLAUDE.md の使い方

CLAUDE.md とは？

プロジェクトフォルダに置く「AIへの常時指示書」です。ここに書いた内容をClaude Codeが常に守ります。

書き方の例

あなたは優秀なビジネスアシスタントです。
回答は必ず日本語で行ってください。
ですます調で、分かりやすく説明してください。
略語や技術用語は避けてください。

活用パターン

ルール:「回答は500文字以内で」「箇条書きで」

役割:「あなたは〇〇の専門家です」

禁止:「英語は使わない」「コードは書かない」

フォーマット:「見出しをつけて」「表形式で」
