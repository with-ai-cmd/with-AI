# クライアント別ディレクトリ

## 構造
```
clients/
├── .template/          # 新規クライアント作成時のテンプレート
│   ├── contracts/      # 契約書（署名済みPDF等）
│   ├── invoices/       # 発行済み請求書（PDF）
│   └── notes/          # メモ・議事録
├── {会社名}/
│   ├── contracts/      # 締結済み契約書
│   ├── invoices/       # YYYY-MM_請求書.pdf
│   └── notes/          # 打ち合わせメモ等
```

## 命名規則
- ディレクトリ名: 会社名（法人格省略可）例: `タイム`, `AIKOMON`
- 契約書: `{契約種類}_{締結日}.pdf` 例: `業務委託契約書_2026-04-01.pdf`
- 請求書: `{YYYY-MM}_請求書.pdf` 例: `2026-04_請求書.pdf`
