# 毛穴が開く本当の原因 — 動画編集指示書

## 基本設定
- **アスペクト比**: 9:16（縦型）
- **推奨尺**: 55〜60秒
- **書き出し**: 1080×1920px / 30fps

---

## カット割り & テロップ

### 0〜3秒【フック】
- **映像**: 出演者が真剣な顔でカメラを見る
- **テロップ**: 「毛穴ケアを頑張ってる人ほど、毛穴が開いてます」
  - 「頑張ってる人ほど」を白、「毛穴が開いてます」を赤太字
  - フォント: 太ゴシック
  - アニメーション: 「毛穴が開いてます」が0.3秒遅れて出現（間を作る）
- **SE**: 不穏な「ドン…」
- **BGM**: 解説系BGM、控えめ（07と同系統）

### 3〜15秒【原因①: 洗いすぎ】
- **テロップ①**: 「原因①」（画面左上、常時表示）
- **テロップフロー**: 「ゴシゴシ洗顔 → 皮脂取りすぎ → 肌"やばい！" → 皮脂ドバッ → 毛穴悪化」
  - 矢印を1つずつ出現
  - 最後の「毛穴悪化」で画面軽く揺れ + 赤フラッシュ
- **テロップ③**: 「毛穴ケアで毛穴を悪化させてる」（赤太字、画面中央にドン）
- **SE**: 「ドバッ」のところで水音

### 15〜25秒【原因②: たるみ】
- **テロップ①**: 「原因②」
- **テロップ②**: 「コラーゲン減少 → 肌たるみ → 丸い毛穴が涙型に」
- **演出**: 丸い毛穴のアイコンが縦に伸びて涙型になる簡易アニメーション（あれば）
- **テロップ③**: 「20代後半から始まる」（黄色、警告感）

### 25〜35秒【原因③: 角栓】
- **テロップ①**: 「原因③」
- **テロップ②**: 「皮脂 + 古い角質 → 角栓 → 毛穴を中から押し広げる」
- **テロップ③**: 「自宅の洗顔じゃ取りきれない」（赤太字）

### 35〜50秒【解決策】
- **テロップ**: 「自宅ケアの"引き算" × プロのケアの"足し算"」
  - 画面中央、大きめ
  - 「引き算」を水色、「足し算」を黄色
  - 間にキラキラの「×」マーク
- **映像**: ハイドロ施術のインサート（あれば）
- **SE**: ポジティブなキラキラ音

### 50〜60秒【CTA】
- **テロップ①**: 「ハイドロフェイシャル 初回3,960円〜」（特大）
- **テロップ②**: 「プロフィールから予約」
- **コメント誘導**: 「毛穴ケアで一番後悔してること教えて」（ピンク文字）
- **小テロップ**: 「※個人差があります」
- **BGMフェードアウト**

## サムネイル案
- 出演者が「えっ！？」という驚き顔
- テキスト: 「毛穴ケア頑張ってる人ほど危ない」（赤文字）

## 注意事項
- 07と編集テイストを統一（解説シリーズ）
- 「原因①②③」の番号を常時左上に表示
- 「毛穴ケアで毛穴を悪化」のテロップが最大のインパクト。大きく、赤く
