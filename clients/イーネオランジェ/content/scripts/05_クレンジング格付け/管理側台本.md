# クレンジング格付け — 管理側台本（ディレクター用）

## 企画意図
- **狙い**: 保存数・信頼構築。「忖度なし」の本音レビューは保存されやすく信頼を獲得できる
- **感情設計**: 「え、あの有名商品が"買わない"！？（驚き）」+「プロの目線すごい（信頼）」
- **想定尺**: 60秒
- **CTA**: 「どんなクレンジングでも毛穴の奥は自宅では限界 → ハイドロで」

## 改善ポイント（旧版から）
- フックを「あの人気商品、プロは絶対買いません」に変更（カリギュラ効果）
- 紹介順を「買わない→平凡→おすすめ→神」の昇順に（ネガティブスタートで引き込む）
- コメント誘導を「あなたが今使ってるクレンジング教えて」に（自分ごと化）

## 出演者への事前共有範囲
- 出演者台本 + 商品リストを渡す
- 格付けは出演者のガチ判断でOK
- 事前に成分を確認してもらう

## 完全台本

### フック（0〜3秒）
映像: 出演者が商品を手に取って首を横に振る
テロップ: **「あの人気クレンジング、プロは絶対買いません」**

**このフックが強い理由**: 「え、どれ？」+ 自分が使ってるかも → 最後まで見る

### 格付け本編（3〜50秒）

**紹介順（ネガティブ→ポジティブの昇順で引き込む）:**

**① 買わない（3〜12秒）**
- **拭き取りシート系全般**
  → 「摩擦が肌にダメージ。毛穴の汚れは落ちない。楽さで選ぶと将来の肌に響きます」
- テロップ: 「拭き取りシート → 摩擦ダメージ + 毛穴汚れ残る」

**② 平凡（12〜20秒）**
- **ビオレ メイク落としパーフェクトオイル**
  → 「落ちるけど必要な皮脂まで持っていく。コスパは良いけど毛穴が気になる方にはオーバーキル」
- テロップ: 「洗浄力強すぎ → 必要な皮脂も取る」

**③ おすすめ（20〜32秒）**
- **カウブランド 無添加クレンジングミルク**
  → 「敏感肌向け。日常メイクなら十分。コスパ良し」
- **ビオデルマ サンシビオ**
  → 「拭き取りだけど肌負担少ない。ただし毛穴の奥までは無理」

**④ 神（32〜45秒）**
- **ファンケル マイルドクレンジングオイル**
  → 「潤いを残しつつ毛穴の皮脂まで落とす。クレンジングの教科書」
- **シュウウエムラ アルティム8**
  → 「洗浄力と保湿のバランスが完璧。ダブル洗顔不要」

### まとめ + ブリッジ（45〜52秒）
「ただ正直に言うと、どんなに良いクレンジングを使っても、毛穴の奥の角栓は自宅ケアだけでは限界がある。それを一気にケアできるのがハイドロフェイシャル。」

### CTA（52〜60秒）
テロップ: **「自宅ケア × プロのケア が最強」**
テロップ: 「ハイドロフェイシャル 初回3,960円〜 プロフィールから予約」
コメント誘導: **「今使ってるクレンジング教えて！プロがコメントで答えます」**
小テロップ: 「※個人差があります」

## 撮影前チェックリスト
- [ ] 全商品を実物で用意
- [ ] 格付けボード（神/おすすめ/平凡/買わない）
- [ ] 出演者に成分を確認してもらう
- [ ] ディレクター想定と違ってもOK（ガチ感を優先）

## 薬機法チェック
- 「将来の肌に響く」→ 一般論としてOK
- 「毛穴の奥の角栓」→ 事実の描写でOK
- 「一気にケアできる」→ 「ケア」は許容範囲
