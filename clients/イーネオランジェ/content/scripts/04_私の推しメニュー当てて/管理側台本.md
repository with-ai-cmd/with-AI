# 私の推しメニュー当てて — 管理側台本（ディレクター用）

## 企画意図
- **狙い**: コメント・フォロー。視聴者が「自分ならこの質問する」と参加感を得られる
- **感情設計**: 「わかった！（推理の快感）」+「やば、ワサビシュー（笑）」
- **想定尺**: 50〜60秒
- **CTA**: 正解のメニュー初回料金

## 改善ポイント（旧版から）
- 質問数を10→7に削減（緊張感UP）
- 罰ゲーム追加（結末が気になる → 最後まで見る動機）
- コメント誘導を「あなたなら何の質問する？」に変更（参加型）

## 出演者への事前共有範囲
- 出演者台本のみ
- Aが引くカードは「ハイパーナイフ」を仕込む
- ワサビシュークリームを見せて緊張感を出す

---

## 完全台本

### フック（0〜3秒）
映像: ワサビシュークリームのアップ → 3人の表情
テロップ: **「7問以内に当てないとワサビシュー。推しメニュー当てゲーム」**

### カード引き（3〜6秒）
Aがカードを引く → 見る → ニヤリ
テロップ: 「カードに書かれたメニューは？」+ カード裏面に「？」

### 質問パート（6〜45秒）

**想定展開（仕込みメニュー: ハイパーナイフ）:**
```
Q1:「顔に使う？」→ No
Q2:「体に使う？」→ Yes    ← ここでボディ系に絞れる
Q3:「痛い？」→ No
Q4:「温かくなる？」→ Yes   ← よもぎ蒸しと迷い始める
Q5:「座って受ける？」→ No  ← よもぎ蒸しが消える
Q6:「機械使う？」→ Yes    ← ハイパーナイフかハイフか
Q7:「ハイパーナイフ！」→ 正解！
```

**面白くなるポイント:**
- Q4「温かくなる？→Yes」で「よもぎ蒸し？」と誤答しそうになる
- Q5で「座って受ける？→No」で「え、じゃあよもぎ蒸しじゃない！？」と焦る
- 残り質問数が減るたびに焦りが増す

### 正解発表（45〜52秒）
B・C: 「ハイパーナイフ！」
Aがカードを見せる → 正解
（不正解の場合: ワサビシュークリーム → リアクション）

### CTA（52〜60秒）
A: 「ハイパーナイフ、1回受けたらわかります」
テロップ: **「ハイパーナイフ 初回6,600円 プロフィールから予約」**
コメント誘導: **「あなたならどんな質問する？コメントで教えて！」**
小テロップ: 「※個人差があります」
