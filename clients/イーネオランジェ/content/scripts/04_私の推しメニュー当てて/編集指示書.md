# 私の推しメニュー当てて — 動画編集指示書

## 基本設定
- **アスペクト比**: 9:16（縦型）
- **推奨尺**: 50〜60秒
- **書き出し**: 1080×1920px / 30fps

---

## カット割り & テロップ

### 0〜3秒【フック】
- **映像**: ワサビシュークリームのアップ → 3人
- **テロップ**: 「7問以内に当てないとワサビシュー」
  - 「ワサビシュー」を緑色太字
  - 位置: 画面上部
- **SE**: 不穏な「ドドドド…」音
- **BGM**: 推理系BGM（軽い緊張感）→ **トレンド音源チェック**

### 3〜6秒【カード引き】
- **映像**: カードを引く → 見る → ニヤリ
- **テロップ**: 画面中央に「？」マーク（大きく）
- **SE**: カードめくり音

### 6〜45秒【質問パート】

**常時表示:**
- 画面右上: 残り質問数カウンター「残り7」→「残り6」...
  - 残り3以下で赤色 + 点滅
- 画面左上: 「確定情報」エリア（判明した事実を蓄積表示）
  - 例: 「顔×」「体○」「痛くない」...
  - 小さめ文字、半透明背景

**各質問の構成:**
1. 質問テロップ: 画面下部に白帯で表示
2. Yes → 緑チェック + ピンポン / No → 赤× + ブッ
3. Aのリアクション（ニヤニヤ、焦り）→ 表情アップ

**特別演出:**
| タイミング | 演出 |
|---|---|
| 誤答しそうになる瞬間 | BGM一瞬止め → 「あぶない！」テロップ |
| 残り3問 | テロップ「残り3問…」赤文字点滅 |
| 残り1問 | ドラムロール → 全員の緊張した顔 |
| 正解の瞬間 | ファンファーレ + 画面キラキラ + 「正解！」特大テロップ |
| 不正解の場合 | ブザー → ワサビシュー → リアクション（スロー演出あり） |

### 45〜52秒【正解発表 + 罰ゲーム（該当時）】
- 正解: ファンファーレ + カード見せ
- 不正解: ワサビシュークリームを食べるB・Cの顔アップ → **スローモーション**
  - SE: 「パクッ」→ 沈黙 → 「うわぁぁぁ！！」
  - **このリアクションが動画最大の見せ場**

### 52〜60秒【CTA】
- **テロップ①**: 「ハイパーナイフ 初回6,600円」（特大、中央）
- **テロップ②**: 「プロフィールから予約」
- **コメント誘導**: 「あなたならどんな質問する？」（ピンク文字）
- **小テロップ**: 「※個人差があります」
- **BGMフェードアウト**

---

## サムネイル案
- Aがカードを隠してニヤニヤ、B・Cが焦り顔、手前にワサビシュー
- テキスト: 「7問以内に当てろ！」

## 注意事項
- 「残り質問数」のカウンターが緊張感を生む。必ず常時表示
- 「確定情報」の蓄積表示は視聴者の推理参加に必須
- 正解でも不正解でも面白い構成（正解→スッキリ / 不正解→罰ゲーム）
