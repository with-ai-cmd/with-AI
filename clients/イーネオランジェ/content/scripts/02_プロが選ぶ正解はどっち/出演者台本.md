# プロが選ぶ正解はどっち？ — 出演者台本

## 企画概要
美容の2択クイズです。2人で対決。間違えると罰ゲーム（激酸っぱいグミを食べる）。

## 出演者
- スタッフ2名（回答者）

## 流れ
1. 2人が向かい合って座る。目の前に激酸っぱいグミ
2. 2択問題が出される → 「せーの」でA/B札を上げる
3. 間違えた方が激酸っぱいグミを食べる
4. 全4問 → 最後に罰ゲーム累計が多い方が負け

## 注意点
- **正解は事前に教えません。ガチで答えてください**
- 間違えた時のグミリアクションを全力でお願いします
- 「プロなのに間違えた…」のリアクションが面白さの肝
- 相手が間違えた時は遠慮なく煽ってOK
