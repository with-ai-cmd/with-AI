# プロが選ぶ正解はどっち？ — 動画編集指示書

## 基本設定
- **アスペクト比**: 9:16（縦型）
- **推奨尺**: 50〜60秒
- **書き出し**: 1080×1920px / 30fps

---

## カット割り & テロップ

### 0〜3秒【フック】
- **映像**: 激酸グミのアップ（0.5秒）→ 2人の緊張した顔
- **テロップ**: 「プロが間違えたら激酸グミ。美容どっちが正解？」
  - 「激酸グミ」を黄色、他は白。太ゴシック
- **SE**: 不穏なドン！音
- **BGM**: テンポ良いバラエティBGM → **投稿時にトレンド音源チェックして差し替え検討**

### 3〜52秒【クイズ4問】

**各問の構成（1問あたり10〜12秒）:**

1. 問題テロップ表示（1秒）
   - 画面上部に問題文
   - 左「A: ○○」/ 右「B: ○○」のスプリット表示

2. 考え中 → 「せーの」→ 札を上げる（3秒）
   - SE: チクタク → ドン！

3. 正解発表（1秒）
   - 正解側: 緑○ + ピンポン
   - 不正解側: 赤× + ブブー

4. **罰ゲームシーン（3〜4秒）← ここが最重要**
   - 不正解者がグミを食べる
   - **表情のアップに切り替え**（引き → 顔アップのカット切り替え）
   - **スローモーション**: 口に入れる瞬間を0.5倍速
   - SE: 「ジュワッ」音を追加
   - リアクション（顔をしかめる、叫ぶ等）は**カットしない**
   - 正解者のドヤ顔 or 笑いもちゃんと映す

5. 解説テロップ（2秒）
   - 画面下部に白帯 + テロップ

**質問ごとの特別演出:**

| Q | 特別演出 |
|---|---|
| Q1 | 初回罰ゲーム → リアクションの基準が見える |
| Q2 | 2連続不正解の場合「また！？」テロップ |
| Q3 | 解説に「よもぎ蒸しが効果的」→ 商品名キラキラ演出 |
| Q4 | 最終問題ドラムロール → 料金をCTAに直結 |

### 52〜60秒【エンディング】
- **映像**: 負けた方が最後のグミを食べる → 2人のリアクション
- **テロップ①**: 「よもぎ蒸し 初回3,300円」（特大、中央）
- **テロップ②**: 「あなたも正解でしたか？」
- **コメント誘導テロップ**: 「何問正解だった？コメントで教えて！」（ピンク文字）
- **小テロップ**: 「※個人差があります」

## サムネイル案
- 1人が酸っぱい顔でグミ食べてる、もう1人がドヤ顔
- テキスト: 「プロが間違えたら罰ゲーム！」

## 注意事項
- **罰ゲームのリアクションは絶対にカットしない**（ここが再生を回す）
- グミを食べる瞬間は必ず顔アップに切り替え
- テンポ重視だが、リアクションは長めに残す（面白さと教育のメリハリ）
- トレンド音源で投稿できるとリーチ倍増
