## コンテンツ体系（5カテゴリ × 各2本 = 10本/バッチ）

毎バッチ、以下の5カテゴリからそれぞれ **2本ずつ** 台本を制作する。

### カテゴリ1: クイズ・対決系
エンタメ × 美容知識。再生数・シェア狙い。

**参考フォーマット（美容クイズ早押し対決）:**
```
出題者: 第○問！[美容に関する問い]
  1. [選択肢A]
  2. [選択肢B]
  3. [選択肢C]
(演出指示: 早押しの動作)
回答者: [回答]
出題者: 正解/残念！
(解説 or リアクション)
...
最終問題で施術メニュー or 料金を自然にCTA
```

**制作ポイント:**
- 早押しボタンは100均でOK
- リアクション大きめ＝面白さUP
- テロップで正解の解説を入れて教育要素を残す
- 最後の問題で価格を自然に出してCTAにする
- バリエーション例: ○×クイズ、3択クイズ、連想ゲーム、格付けチェック

### カテゴリ2: ゲーム・バラエティ系
スタッフの人柄が出るエンタメ。フォロー・コメント狙い。

**参考フォーマット（NGワードを言わせろ）:**
```
(セットアップ: スタッフの額にNGワードを貼る)
スタッフA: [質問で相手を誘導]
スタッフB: [NGワードを避けながら回答]
(テロップで状況を補足: 「言いそうで言わない」「必死に回避中」)
...
最後にNGワードを言ってしまう → オチ
(テロップで施術の実感に繋げる)
```

**参考フォーマット（ワードウルフ）:**
```
(スタッフ3人がカードを見る。2人は同じメニュー、1人だけ違う)
各自がヒントを出し合う → 人狼を推理
投票 → 正体バラし
(メニューの特徴が自然に伝わる構成)
テロップ: 「ワードは概要欄に書いてます」
```

**制作ポイント:**
- スタッフの自然なやり取り＝サロンの雰囲気が伝わる
- ゲーム内で施術メニューの特徴が自然に出る設計にする
- バリエーション例: NGワード、ワードウルフ、イントロクイズ、利き○○、ジェスチャーゲーム

### カテゴリ3: 格付け・ランキング系
プロの目線で市販品をガチ評価。保存数・信頼獲得狙い。

**参考フォーマット（4段階格付け）:**
```
評価ランク: 神 / おすすめ / 平凡 / 買わない

[商品名]
→「[プロ視点のコメント。毛穴への影響を軸に]」

※ 全商品をこのフォーマットで評価
```

**制作ポイント:**
- 「神」「買わない」の両極端が一番バズる
- 毛穴・温活の視点で評価すると専門性が出る
- 高価格帯ブランドに忖度しない＝信頼性UP
- バリエーション例: 日焼け止め、美容液、クレンジング、化粧水、ボディクリーム、入浴剤

### カテゴリ4: 教育・解説系
専門知識で信頼構築。保存数・フォロー転換狙い。

**参考フォーマット（60秒解説）:**
```
【フック 0〜3秒】
テロップ: 「[衝撃 or 共感の一文]」
「[問題提起のナレーション]」

【解説 3〜45秒】
「[仕組みをわかりやすく解説]」
(テロップ: 「[要点を図解的に]」)
「[原因と結果の因果関係を説明]」
(テロップ: 「[A→B→C の流れ]」)

【解決策 45〜55秒】
「[施術メニューがどう解決するかを説明]」
(テロップ: 「[解決の流れ]」)

【CTA 55〜60秒】
「[行動を促す一言]」
テロップ: 「初回5,500円 ✦ プロフィールから予約」
```

**制作ポイント:**
- フックは「え、そうなの？」と思わせる意外性
- 解説は専門用語を使わず「体のデリバリーシステム」のような比喩を使う
- テロップで「A→B→C」の因果関係を図解的に示す
- 最後の10秒でメニューに自然に繋げる
- テーマ例: 毛穴×ハイドロ、冷え×よもぎ蒸し、むくみ×ハイパーナイフ、乾燥×保湿、紫外線×ケア

### カテゴリ5: ビフォーアフター・施術紹介系
施術効果の可視化。予約CV狙い。

**フォーマット:**
```
【フック 0〜3秒】
テロップ: 「[施術名]やったらこうなった」or「○ヶ月通った結果」

【施術シーン 3〜30秒】
(施術の様子をテンポよく。ASMR的な心地よさも意識)
ナレーション or テロップで施術の解説

【結果 30〜50秒】
ビフォーアフター比較
お客様の感想（許可済み）
テロップで変化のポイントを強調

【CTA 50〜60秒】
テロップ: 「初回○,○○○円 ✦ プロフィールから予約」
```

**制作ポイント:**
- 薬機法注意: 「※個人の感想です」「※効果を保証するものではありません」を必ず記載
- 施術音・ASMR要素は完視聴率を上げる
- 照明を統一してビフォーアフターの信頼性を保つ

## 出力体系（3ドキュメント分離）

1本の企画につき、以下の **3種類** を必ずセットで作成する。

### A. 出演者用台本
**目的**: 出演スタッフに渡す台本。答え・オチ・仕掛けは絶対に書かない。
**なぜ分離するか**: クイズの答えやゲームのオチを知ってしまうとリアクションが不自然になる。

含める情報:
- 企画名・自分の役割（出題者/回答者/プレイヤー等）
- 大まかな流れ（「クイズが出されるので早押しで答えてください」等）
- NGワード（薬機法的に言ってはいけない表現）
- 服装・見た目の指示
- テンション・リアクションの方向性（「大げさめに」「自然体で」等）

含めない情報:
- クイズの正解
- ゲームのオチ・仕掛け
- 他の出演者の役割詳細
- 編集で入れるテロップ内容

### B. 管理側台本（ディレクター用）
**目的**: 企画の全体像。正解・オチ・演出意図・撮影指示を全て含む完全版。

含める情報:
- 企画の意図・狙い（再生数/保存/CV等）
- 完全な台本（正解・オチ含む）
- 各出演者への指示メモ（「この人にはここまで伝える」等）
- 撮影前の準備チェックリスト
- 想定尺
- CTA設計
- 薬機法チェック済みの確認

### C. 動画編集指示書
**目的**: 編集者が迷わず仕上げられる指示書。

含める情報:
- カット割り（どこで切り替えるか）
- テロップ: 表示テキスト / タイミング / 位置 / フォントサイズ感
- SE・効果音の指示（正解音、ブザー、ドラムロール等）
- BGM指示（テンション感、参考曲のジャンル）
- エフェクト指示（ズーム、スロー、画面揺れ等）
- サムネイル案
- 書き出し設定（縦9:16、推奨尺）

## 制作フロー

```
1. テーマ決定（アナリストの勝ちパターン分析 + 季節トレンドを参考）
2. 5カテゴリ × 2本 = 10企画の台本を一括作成
3. 各企画 × 3ドキュメント = 30ファイルを生成
4. CEO確認 → フィードバック反映
5. 出演者台本を撮影前にスタッフへ共有
6. 撮影後、編集指示書を編集者へ共有
7. コピーライターに投稿文作成を依頼
```

## 制作ルール
1. **フック最優先**: 冒頭3秒で離脱を防ぐ。疑問形・衝撃・共感のいずれかを使う
2. **テンポ重視**: ショート動画は情報密度が命。無駄なカットは削る
3. **ビフォーアフター**: エステの効果を視覚的に伝える構成を積極的に使う
4. **信頼性**: 薬機法に抵触する表現（「治る」「確実に」等）は絶対に使わない
5. **ブランドトーン**: 高級感がありつつも親しみやすい。押し売り感を出さない
6. **CTA統一**: 最後は「初回○,○○○円 ✦ プロフィールから予約」で統一
7. **毛穴 × 温活を軸に**: 全コンテンツ、イーネのUSPに繋がるよう設計する
8. **3ドキュメント分離を厳守**: 出演者台本に答え・オチを絶対に書かない

## 薬機法NGワード（例）
- 治る、治療、医療級、確実に○○になる、副作用なし
- 代わりに: 「目指す」「ケアする」「サポート」「実感」等を使用
