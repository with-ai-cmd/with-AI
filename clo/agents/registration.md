# 登記手続きエージェント（Registration Agent）

## 概要
商業登記に関する書類の自動生成・チェック・手続きガイドを行うエージェント。
GVA法人登記の書式に準拠した書類一式をPDFで出力する。

## 対応する登記種別

| 種別 | テンプレート | 状態 |
|---|---|---|
| 役員変更（就任） | `generate_officer_change.py` | 実装済み |
| 役員変更（辞任） | 未実装 | 計画中 |
| 役員変更（重任） | 未実装 | 計画中 |
| 本店移転 | 未実装 | 計画中 |
| 目的変更 | 未実装 | 計画中 |
| 商号変更 | 未実装 | 計画中 |
| 増資 | 未実装 | 計画中 |

## 役員変更（就任）の生成フロー

```
ユーザー入力（役員名、住所、役職、株主構成、日付）
  ↓
① config.json を生成
  ↓
② generate_officer_change.py --config config.json
  ↓
③ 全8書類をPDF出力
  ↓
④ チェックリストを表示
  ↓
⑤ 必要に応じて修正・再生成
```

### 生成される書類

| # | 書類 | 説明 |
|---|---|---|
| 01 | 取締役決定書（招集決定等） | 株主総会決議省略の決定 |
| 02 | 提案書 | 株主への議案提案（案内文+提案内容） |
| 03 | 同意書 | 株主ごとに1通（議案への同意） |
| 04 | ★株主総会議事録（みなし） | みなし決議の議事録 |
| 05 | ★株主リスト | 商業登記規則61条に基づく株主一覧 |
| 06 | ★就任承諾書 | 新任役員ごとに1通 |
| 07 | ★登記申請書 | 法務局提出用（別紙・収入印紙台紙含む） |
| 08 | 必要書類のご案内 | 印鑑証明書等の案内 |

※ ★印は法務局提出必須書類

## チェックリスト（生成後に表示）

生成後、ユーザーに以下を確認してもらう:

- [ ] 会社名・本店所在地が登記簿と一致しているか
- [ ] 会社法人等番号が正しいか
- [ ] 新任役員の氏名・住所が住民票と一致しているか
- [ ] 株主構成（持株数・割合）が最新か
- [ ] 管轄法務局が正しいか
- [ ] 登録免許税の金額が正しいか（資本金1億円以下: 1万円、超: 3万円）
- [ ] 各書類の日付が整合しているか
- [ ] 印鑑証明書を取得したか（発行日から3ヶ月以内）

## 注意事項

1. **法的書類の正確性が最優先** — 氏名・住所は住民票・登記簿の表記と完全一致させる
2. **日付の整合性** — 決定日 < 同意期限 = 決議日 ≤ 就任日 ≤ 証明書作成日
3. **登録免許税** — 役員変更は1万円（資本金1億円以下）、3万円（超）
4. **印鑑証明書** — 代表取締役就任には個人の実印+印鑑証明書が必須
5. **本人確認** — 取締役のみの就任でも印鑑証明書が必要（取締役会非設置会社の場合）
