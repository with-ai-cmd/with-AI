# Chalk-Board HTML スライド — スタイル定義

黒板風HTMLスライドのデザインシステム。
チョークで書いたような温かみのある教育的デザイン。

## デザインコンセプト

「教室の黒板で行われた、わかりやすい授業」
— チョークの質感、黒板消しの存在感、教室のノスタルジーを再現する。

## チョークテキストエフェクト

すべてのテキストにチョークの質感を与える:
- `text-shadow` で微かなにじみを表現
- タイトル: 太めのチョーク（letter-spacing広め）
- 本文: 標準のチョーク
- 注釈: 細めのチョーク

## スライドサイズ

- アスペクト比: 16:9
- 最大幅: 960px
- 最大高さ: 540px
- レスポンシブ: ビューポートに合わせてスケール

## ナビゲーション

- **前へ/次へボタン**: スライド下部、チョーク風テキスト
- **ドットインジケーター**: チョーク白の丸ドット
- **プログレスバー**: スライド上部、チョーク黄の細いバー
- **キーボード**: 左右矢印キー、スペースで次へ
- **タッチ**: スワイプ対応

## 黒板消しデザイン

表紙スライドの右下に黒板消しのSVGイラストを配置:
- 木製の持ち手（暖かい茶色）
- 黒いフェルト面
- チョーク粉が付着した質感
- 消し跡（薄いぼかし領域）を黒板面の一部に表示

## チョーク囲み（プロンプトコピー用）

プロンプト例やコード例は「チョーク囲み枠」スタイルで表示:
- チョーク線の手描き風ボーダー（dashed）
- 背景は黒板面より少し明るい半透明
- 左上にラベル（チョーク黄で小さく表示）
- 右上にコピーボタン（チョーク白）
- チョーク色でテキスト表示

## ページタイプ

| タイプ | 用途 |
|---|---|
| cover | 表紙。タイトル + チョーク落書きSVG + 黒板消し |
| goal | 導入。学習目標を表示 |
| content | 本編。テキスト + チョーク図解 |
| comparison | 比較。左右で2つの概念を対比 |
| prompt-example | プロンプト例。チョーク囲み + コピーボタン |
| summary | まとめ。チェックマーク付き要点リスト |
| glossary | 用語集。チョーク枠カード型レイアウト |

## 黒板の質感表現

- 微かなノイズテクスチャ（CSS radial-gradient で表現）
- 薄い消し跡（以前のチョークが残った感じ）
- 黒板の木枠は非表示（スライド面のみ）
- 全スライドで同じ黒板の質感を維持
