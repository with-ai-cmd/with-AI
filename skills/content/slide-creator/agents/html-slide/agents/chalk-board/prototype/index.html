プロンプトとは？
プロンプトとは、生成AIに入力する指示文のことです。
レストランの注文に例えると、メニューを見て「カルボナーラをください」と伝えるのがプロンプト。AIはシェフとして、その注文どおりの料理を作ります。
ひとことで言うと
プロンプト = AIへの「お願い」や「質問」をテキストで書いたもの

プロンプトの構成要素
1. 指示（Instruction）
AIに何をしてほしいかを伝える部分。
例：「要約してください」「翻訳して」
2. 文脈（Context）
背景情報や前提条件。
例：「初心者向けに」「ビジネスメールとして」
3. 入力データ（Input）
処理してほしい対象のテキストやデータ。
例：「以下の文章を〜」
4. 出力形式（Output Format）
どんな形で答えてほしいか。
例：「箇条書きで」「表形式で」「JSON形式で」
この4つを意識するだけで、プロンプトの質が格段に上がる

良いプロンプト vs 悪いプロンプト
悪いプロンプト
PROMPT AIについて教えて
曖昧で範囲が広すぎる。AIは何を答えればいいかわからない。
VS
良いプロンプト
PROMPT 中学生にもわかるように、 AIの定義を200文字以内で 説明してください。 具体例を1つ含めてください。
対象・形式・分量・条件が明確。期待どおりの回答が返る。

役割と背景情報を伝える
役割（ロール）を与える
「〜の専門家として答えて」と指定すると、回答の視点や深さが変わります。
PROMPT あなたは小学校の先生です。 小学3年生にもわかるように、 「AI」とは何かを説明してください。
背景情報（コンテキスト）
「なぜこの質問をしているか」を伝えると、的外れな回答が減ります。
PROMPT 来週の社内会議で、新入社員10名に 情報セキュリティの基本を説明します。 15分のプレゼン構成案を作ってください。

プロンプト作成 7つのコツ
  • 具体的に書く — 5W1Hを意識
  • 出力形式を指定 — 箇条書き・表形式など
  • 役割を与える — 「〜の専門家として」
  • 背景情報を伝える — 目的・対象者・場面
  • 条件を設定 — 文字数・トーン・除外条件
  • 一つずつ指示 — 複雑なタスクは分割
  • 例文を示す — Few-Shotの考え方

用語集
プロンプト (Prompt)
生成AIに入力する指示や質問のテキスト。AIの出力を左右する最も重要な要素。
プロンプトエンジニアリング
プロンプトを工夫・最適化してAIから望みどおりの出力を引き出す技術。
コンテキスト (Context)
AIへの指示における背景情報や前提条件。対象者・目的・場面など。
Few-Shot プロンプティング
AIに1〜数個の例文を示してから回答させる手法。出力の形式や雰囲気を揃えやすい。
5W1H
Who/What/When/Where/Why/How。具体的なプロンプトを書くための基本フレームワーク。
出力形式 (Output Format)
箇条書き・表形式・JSON等、AIにどんな形で答えてほしいかを指定すること。
