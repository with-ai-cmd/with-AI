# AI Seminar — AIセミナースライド自動生成エージェント

AIセミナーや勉強会の内容テキストを入力するだけで、**近未来的なHTMLスライド**を自動生成するエージェント。

## 概要

親エージェント `html-slide` のデザインシステムを継承し、AI・テクノロジーセミナーに特化したスライドを生成する。

## 利用方法

```
/slide-creator html-slide ai-seminar

（その後、セミナーの内容テキストを貼り付ける）
```

## 自動処理フロー

```
セミナー内容テキスト入力
    ↓
① 構造解析：見出し・セクション・箇条書きを自動パース
    ↓
② スライド分割：内容をスライド単位に自動分割
    ↓
③ イラスト選定：各スライドに最適なSVGイラストを割り当て
    ↓
④ HTML生成：html-slide エージェントのデザインシステムで出力
    ↓
⑤ ファイル保存：デスクトップに出力
```

## セミナー特化機能

### 自動イラスト割り当て

テーマに応じて最適なアイソメトリックSVGを自動生成：

| テーマキーワード | 割り当てイラスト |
|---|---|
| AI / 人工知能 | ロボット・ニューラルネットワーク図 |
| 自動化 / ワークフロー | フローチャート・歯車 |
| データ / 分析 | グラフ・ダッシュボード |
| チーム / 組織 | 組織図・並列エージェント |
| クラウド / API | サーバー・接続図 |
| セキュリティ | シールド・ロック |

### 自動セクション構成

入力テキストから以下を自動検出：
- `【導入】` → イントロスライド
- `【セクション】` → テーマ別スライド群
- `■ スライドN：` → 個別スライド分割
- `QR` → 最終スライドにQRコード配置

### デフォルト設定

| 項目 | デフォルト値 |
|---|---|
| カラーテーマ | Cyberpunk Dark（黒背景・ネオンカラー） |
| フォント | Orbitron（見出し）+ Noto Sans JP（本文） |
| アニメーション | パーティクル + グロー + フェード |
| ナビゲーション | 矢印キー + タッチ + ドット |
| イラスト方式 | ミックス（インラインSVG + CSS図形） |
| QRコードAPI | api.qrserver.com |

### QRコード

最終スライドにQRコードを自動配置。URLが提供されない場合はスキップ。

## 入力例

```
プレゼンタイトル：
Claude Codeで話題のClaudeを深掘り！

【導入】（約2分）
■ スライド1：タイトル＆自己紹介
画面: タイトルと登壇者のプロフィール。
■ スライド2：本日のゴール
画面: 大きな文字で「AIは『相談相手』から『優秀な部下』へ変わった」

【セクション1：Claudeとは？】
■ スライド3：話題のAI「Claude」とは？
...
```

## 出力

- **ファイル**: `index.html`（単一ファイル・CSS/JS内蔵）
- **保存先**: `~/Desktop/ai-seminar-slides/` またはユーザー指定ディレクトリ
- **ブラウザ**: Chrome/Safari/Firefox で動作確認済み想定
