ChatGPTって、そもそも何？
ChatGPTは、米OpenAI社が開発した対話型の生成AIサービスです。

人間と自然な会話ができるのが最大の特徴で、テキストで質問や指示を入力すると、AIが文章で回答します。

レストランに例えると、メニューが無限にあるレストランのようなもの。「こんな料理を作って」とお願いすれば、AIシェフがその場で作ってくれます。

2022年11月にリリースされ、わずか2ヶ月で1億ユーザーを突破。2026年現在、世界で数億人が利用しています。

具体的にどんなことができるの？
ChatGPTでできることは大きく分けて7つあります：
  1. 文章の生成・下書き — メール、レポート、ブログ記事
  2. 要約 — 長い文書を短くまとめる
  3. 翻訳 — 多言語間の翻訳
  4. 質疑応答 — 知識に基づいた回答
  5. コード生成 — プログラムの作成・デバッグ
  6. 画像の生成・読み取り — DALL·E 3による画像生成
  7. Web検索 — リアルタイムの情報取得
特に2025〜2026年にかけて、Deep ResearchやAdvanced Voiceなど高度な機能が追加されました。

GPT-4oとかo3とか、モデルの違いがよくわからない
主なモデルを整理しますね：
モデル比較
GPT-4o → 高速・マルチモーダル（テキスト+画像+音声） GPT-4o mini → 軽量・高速・コスパ重視 o3 → 高度な推論（数学・科学・コーディング） o4-mini → 推論モデルの軽量版
「o」が付くモデルは推論特化型、「GPT-4o」は汎用型です。

車で例えると、GPT-4oは万能なセダン、o3は難問を解くF1マシン。

ChatGPTを使うときに気をつけることは？
5つの注意点を覚えておきましょう：
  1. ハルシネーション — AIはもっともらしいウソをつくことがある。重要な情報は必ずファクトチェック
  2. 個人情報を入力しない — 氏名・住所・パスワードなどの機密情報は避ける
  3. 著作権に注意 — AI生成物をそのまま公開する場合は権利関係を確認
  4. 最新情報の限界 — モデルの学習データには期限がある
  5. 回答は出発点 — AIの答えは「下書き」。人間が確認・編集して完成
