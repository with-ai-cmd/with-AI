# ChatGPT-UI HTML スライド — スタイル定義

ChatGPTの画面UIを忠実に再現したHTMLスライドのデザインシステム。
教材の内容がチャットの会話形式で進行し、実際にChatGPTを使っているような体験で学べる。

## デザインコンセプト

「ChatGPTを使いながら学んでいる体験」
— 実際のChatGPT画面を忠実に再現し、ユーザーメッセージとAI回答の会話形式で教材が進む。

## UI構成

### サイドバー（左側・装飾用）
- 幅: 180px（装飾的。実際のナビ機能はスライドナビで代替）
- 「New chat」ボタン風
- チャット履歴リスト風（章内のレッスンタイトルを表示）
- 現在のレッスンをハイライト

### メインチャットエリア
- 会話が上から下に流れる
- ユーザーメッセージ: 右寄せ、角丸バブル、ユーザーアイコン
- AIメッセージ: 左寄せ、バブルなし、ChatGPTアイコン
- メッセージ間に適度な間隔

### 入力欄（下部・装飾/プロンプト例表示用）
- 角丸のテキスト入力風
- 送信ボタン（↑アイコン）
- プロンプト例をここに表示 + コピーボタン

## メッセージスタイル

### ユーザーメッセージ
- 右寄せ配置
- 背景: `--gpt-user-bubble`
- 角丸: 20px
- アバター: 紫の丸に「U」

### AIメッセージ
- 左寄せ配置
- 背景: 透明（メイン背景と同じ）
- ChatGPTアイコン: 緑丸にスパークルマーク
- 太字・箇条書き・コードブロック対応

## ページタイプ

| タイプ | 用途 |
|---|---|
| welcome | 表紙。ChatGPTのウェルカム画面風 |
| chat | 会話形式。ユーザーQ → AI回答で内容を説明 |
| comparison | 2つのチャット例を左右で比較 |
| code-block | プロンプト/コード例。入力欄風にコピーボタン付き |
| summary | まとめ。チェックマーク付きAI回答 |
| glossary | 用語集。AIが一覧で回答する形式 |
