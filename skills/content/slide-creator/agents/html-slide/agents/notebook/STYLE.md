# Notebook HTML スライド — スタイル定義

大学ノート風HTMLスライドのデザインシステム。
罫線ノートにペンとマーカーで書いたような、親しみやすい教育的デザイン。

## デザインコンセプト

「自分のノートを見返しているような学習体験」
— 罫線、蛍光マーカー、ペン書き、付箋、マスキングテープ。手元のノートの温かみを再現する。

## ノートの質感表現

- 横罫線（約28px間隔、薄い水色）
- 左端に赤い縦マージン線（1本）
- 紙のテクスチャ（微かなクリーム色グラデーション）
- ページ左側にリング穴を模した装飾（6個の小さな丸）
- 右下にページ番号（手書き風）

## 蛍光マーカーエフェクト

重要テキストを蛍光マーカーで引いたように表示:
- 線は少し斜め（transform: skewX(-2deg)）
- テキストの背後に半透明カラーバー
- はみ出し感（padding左右を少し広め）

## 付箋デザイン

- 上部にマスキングテープの装飾（斜め貼り）
- 微かな影（box-shadow）
- 少し傾いた角度
- 4色をローテーション

## ページタイプ

| タイプ | 用途 |
|---|---|
| cover | 表紙。タイトル + ノート風イラスト + デコレーション |
| goal | 導入。学習目標を表示 |
| content | 本編。罫線上にテキスト + 図解 |
| comparison | 比較。付箋スタイルで左右対比 |
| prompt-example | プロンプト例。コード枠 + コピーボタン |
| summary | まとめ。チェックボックス付き要点リスト |
| glossary | 用語集。カード型レイアウト |
