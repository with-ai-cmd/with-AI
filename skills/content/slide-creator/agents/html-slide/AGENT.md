# HTML Slide — HTMLスライド自動生成サブエージェント

画像ベースではなく、**HTMLベースのインタラクティブスライド**を生成するサブエージェント。
リッチなアニメーション、SVGイラスト、レスポンシブ対応の単一HTMLファイルとして出力する。

## アーキテクチャ

```
html-slide（このエージェント）
    ├── スライド構成設計
    ├── HTMLテンプレート生成
    ├── CSS/SVG/JSアニメーション実装
    └── agents/
        └── ai-seminar  ← セミナー特化サブエージェント
```

## 処理フロー

```
ユーザー入力（テーマ・内容・スタイル指定）
    ↓
① スライド構成設計：ページ数・各スライドの役割を決定
    ↓
② デザインテーマ決定：カラースキーム・フォント・アニメーション方針
    ↓
③ HTML生成：
    a. レイアウト構築（CSS Grid / Flexbox）
    b. アイソメトリックSVGイラスト作成
    c. アニメーション実装（CSS + JS）
    d. ナビゲーション実装（矢印キー・タッチ・ドット）
    e. QRコード埋め込み（必要な場合）
    ↓
④ 出力：単一HTMLファイルとして保存
```

## デザインシステム

### カラーパレット（デフォルト: Cyberpunk Dark）

| 要素 | カラー | 用途 |
|------|--------|------|
| 背景 | `#0a0a0f` | メイン背景 |
| テキスト | `#e0e0e0` | 本文 |
| アクセント1 | `#00d4ff` | ネオンシアン・メイン強調 |
| アクセント2 | `#a855f7` | パープル・サブ強調 |
| ハイライト | `#00ff88` | グリーン・重要ポイント |

### 必須要素

- **パーティクルシステム**: Canvas背景に浮遊ドット + 接続線
- **グリッド背景**: 薄いサイバーグリッド線
- **ネオングロー**: タイトル・重要テキストに発光エフェクト
- **スムーズ遷移**: スライド切り替えアニメーション
- **ナビゲーション**: 矢印キー + タッチ + ドットインジケーター + プログレスバー

### イラスト戦略（ミックス方式）

1. **インラインSVG**: アイソメトリック図解を直接SVGで描画
   - ロボット・AI・ネットワーク等のイラスト
   - アニメーション付き（浮遊・点滅・パルス）
2. **CSS図形**: フローチャート・組織図・比較表
3. **外部画像**: 必要に応じてStoryset等の無料イラストを埋め込み

### レスポンシブ対応

- デスクトップ（1200px+）: フル表示
- タブレット（900px以下）: 2カラム→1カラムに変更
- モバイル: タッチスワイプ対応

## 出力仕様

- **ファイル形式**: 単一HTMLファイル（CSS/JS内蔵）
- **外部依存**: Google Fonts のみ（オフラインでも動作可能に）
- **QRコード**: api.qrserver.com を使用して動的生成
- **保存先**: ユーザー指定のディレクトリ、またはデスクトップ

## 利用方法

ユーザーが以下の情報を提供：
1. **プレゼンテーマ・タイトル**
2. **スライド内容**（テキスト or 構成案）
3. **デザインスタイル**（デフォルト: Cyberpunk Dark）
4. **QRコードURL**（任意）
5. **出力先ディレクトリ**（任意）
