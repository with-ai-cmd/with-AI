# ホワイトボードスタイル デザインガイドライン

## デザインコンセプト

「ホワイトボードで行われたエネルギッシュな戦略セッション」
— 手書きマーカーの温かみと、ブレインストーミングのライブ感を再現する。

## 世界観

- 物理的なホワイトボードに書かれたようなスライド
- フレームなし（白背景が端まで広がる）
- ホワイトボード面は清潔感のある白（光の反射は控えめ）
- きっちり整列しすぎず、有機的で少し雑なレイアウト
- リアルなブレインストーミングセッションの雰囲気

## カラーパレット（4色のみ・厳格遵守）

| 用途 | 色 |
|---|---|
| 背景 | 白（ホワイトボード面） |
| メインテキスト・線・枠線 | 黒マーカー |
| アクセント・見出し装飾 | 青マーカー |
| 強調・重要マーク | 赤マーカー |
| 補足・チェックマーク | 緑マーカー |

**禁止事項：**
- 上記以外の色の使用（黄色・オレンジ・紫・ピンク等）
- グラデーション
- 半透明・ソフトシェーディング
- パステルカラー

## テキストスタイル

すべてのテキストは「手書きホワイトボードマーカー」で書いたように見えること。

| 用途 | 大きさ | 色 |
|---|---|---|
| 表紙タイトル | 最大（スライド内で一番大きい） | 黒 |
| ページタイトル | 大きめ | 黒 |
| サブタイトル | 中くらい | 青 |
| 本文・箇条書き | 読みやすい標準サイズ | 黒 |
| 注釈・補足 | 小さめ | 赤 or 緑 |
| 左上ヘッダー | 小さめ | 青 |
| ページ番号 | 最小 | 黒 |

## 固定レイアウト要素

### 左上ヘッダー（全スライド共通）
- 位置: 左上の角
- 内容: 章ラベル（例: `1章 AI基礎知識`）
- 色: 青マーカー
- サイズ: 小さめ

### ページ番号（全スライド共通）
- 位置: 右下
- 形式: `{ページ番号}/{総ページ数}`
- 手書きスタイル

## ページタイプ別デザイン方針

### 表紙（P1）
- 最もシンプル。中央にタイトルのみ
- 周囲に関連する手描きdoodleを散らす
- エネルギッシュで期待感のある雰囲気

### 導入（P2）
- 一文要約を吹き出しや枠で目立たせる
- 学習目標を箇条書き
- 学びに関連するイラスト（電球、本、虫眼鏡）

### 本編（P3-P8）
- タイトル + 赤いアンダーライン
- 要点3〜5個、手描きアイコン付き
- 右側または下部に関連する図解/イラスト
- 重要語は赤丸で囲む

### まとめ（P9）
- 緑チェックマーク付きの要点リスト
- 達成感を感じるデザイン（星、トロフィーdoodle）

### 用語集（P10）
- 手描き枠のカード型レイアウト（2列）
- キーワードは青太字、説明は黒通常
