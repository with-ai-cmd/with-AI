# white-board-slide サブエージェント

ホワイトボード風の手書きマーカースタイルで画像スライドを生成するサブエージェント。
nanobanana-slide エージェントから呼び出される。

## 役割

nanobanana-slide から受け取ったスライド構成（`slide_structure.json`）を元に、
ホワイトボード風のプロンプトを生成し、画像スライドを作成する。

## 処理フロー

```
nanobanana-slide から呼び出し
    ↓
① スタイル読み込み: 同ディレクトリの STYLE.md + prompt-template.md を読む
    ↓
② プロンプト生成: テンプレートにコンテンツを流し込み、10枚分のプロンプトを作成
    → image_prompts.json として保存
    ↓
③ 結果を nanobanana-slide に返す（画像生成・PDF化は親が実行）
```

## スタイルファイル

| ファイル | 役割 |
|---|---|
| `STYLE.md` | 厳密なスタイルルール（色・フォント・統一性・禁止事項） |
| `prompt-template.md` | 画像生成プロンプトのテンプレート（6種レイアウト） |
| `design-guide.md` | デザインガイドライン（人間向けドキュメント） |

## スタイルの特徴

- **コンセプト**: 教師がホワイトボードに書いた授業ノート
- **カラー**: 黒・青・赤・緑の4色マーカーのみ
- **フレーム**: なし（白背景が端まで広がる）
- **テキスト**: 丸みのある丁寧な手書き。文字は必ず水平
- **統一性**: 全10枚が同一人物・同一マーカーセットで書かれた見た目

## 6種類のレイアウト

| タイプ | 名前 | 用途 |
|---|---|---|
| A | イラスト中心 | 大きな手描きイラストで概念を視覚的に説明 |
| B | 箇条書き+小イラスト | 左にコンパクトなポイント、右に補足スケッチ |
| C | 比較・対比 | T字チャートで2つの概念を左右比較 |
| D | フローチャート | 矢印でステップ・プロセスをつなぐ |
| E | 中心概念+放射 | マインドマップ風に構成要素を放射状展開 |
| F | 大きな図解 | ベン図・ピラミッド・同心円など+キャプション |

## プロンプト生成の厳守事項

`STYLE.md` の禁止事項を必ず守ること：
- フォント名をプロンプトに含めない
- ポイント数をプロンプトに含めない
- カラーコードをプロンプトに含めない
- 色は色名のみ（black, blue, red, green）
- サイズは相対表現のみ（very large, large, medium, small, tiny）
- マーカー太さ: タイトル=thick chisel-tip、本文=standard、注釈=fine-tip
- テキストサイズ比率: タイトル:本文:注釈 = 3:2:1
- 日本語: 丸みのある丁寧な手書き
- 文字方向: すべて水平
