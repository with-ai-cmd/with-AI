# White-Board スタイル定義

nanobanana-slide サブエージェントが使用するホワイトボード風スライドのスタイル定義。

## 目的

画像生成AIに渡すプロンプトの「見た目ルール」を厳密に定義する。
このファイルの内容はプロンプト生成時に参照され、全10スライドで完全に統一された外観を実現する。

## 絶対禁止事項

以下は画像内に絶対に含めてはならない：

- フォント名（Permanent Marker, Caveat, Kalam, Arial 等）
- ポイント数（14pt, 18pt, 48pt 等のサイズ指定）
- カラーコードのテキスト表示（#000000, #FF0000 等）
- プロンプト指示文そのもの（"size equivalent to", "similar to" 等）
- デジタル的なフォント・UIパーツ

## 統一性の核心ルール

### 同一人物ルール
全10スライドは**同一人物が同じマーカーセットで、同じホワイトボードに連続して書いた**ものとして生成する。
- 筆圧、文字の癖、マーカーの太さ、余白の取り方がすべて統一されていること
- 1枚目と10枚目で「別の人が書いた」ように見えてはならない

### 手書きスタイルの具体的指定
- **日本語**: 丸みのある丁寧な手書き。角ばったゴシック体ではなく、少し丸い親しみやすい文字。教師がホワイトボードに丁寧に書くようなスタイル
- **英語・数字**: 丸みのあるブロック体。読みやすさ最優先
- **マーカーの太さ**:
  - タイトル = 太いマーカー（chisel tip marker, broad strokes）
  - 本文 = 中太マーカー（standard marker）
  - 注釈・ページ番号 = 細いマーカー（fine tip marker）
- **文字の傾き**: すべて水平に整列。斜めに書かない
- **文字間隔**: 均等で読みやすい。詰めすぎない

## フレームなし

スライドにフレーム（枠）は付けない。白背景が画像の端まで広がる。

## カラーパレット — 4色のみ厳守

ホワイトボードマーカーの4色 + 白背景のみ使用可能：

| 用途 | 色 | マーカータイプ |
|---|---|---|
| 背景 | 白（ホワイトボード面） | — |
| メインテキスト・線・枠線 | 黒 | 太・中太・細を使い分け |
| アクセント・見出し装飾 | 青 | 中太〜太 |
| 強調・重要マーク | 赤 | 中太 |
| 補足・チェックマーク | 緑 | 中太 |

### 厳格ルール
- 上記5色以外の色は一切使用禁止
- グラデーション禁止
- 半透明・ソフトシェーディング禁止
- パステルカラー禁止
- 黄色・オレンジ・紫・ピンク等は使用禁止
- すべての描画はソリッドなマーカーインクの見た目
- **マーカーのインク濃度は全スライドで統一**（薄くなったり濃くなったりしない）

## テキストスタイル — サイズは相対指定のみ

すべてのテキストは「手書きホワイトボードマーカー」で書かれたように見えること。

| 用途 | 大きさの指示方法 | 色 | マーカー太さ |
|---|---|---|---|
| 表紙タイトル | スライド内で最も大きいテキスト | 黒 | 太い |
| ページタイトル | 大きめのテキスト | 黒 | 太い |
| サブタイトル | 中くらいのテキスト | 青 | 中太 |
| 本文・箇条書き | 読みやすい標準サイズ | 黒 | 中太 |
| 注釈・補足 | 小さめのテキスト | 赤 or 緑 | 細い |
| 左上ヘッダー | 小さめのテキスト | 青 | 細い |
| ページ番号 | 小さいテキスト | 黒 | 細い |

### テキストサイズの比率（統一性のため）
全スライドで以下の比率を維持する：
- タイトル : 本文 : 注釈 = **3 : 2 : 1** の比率
- この比率が全10スライドでブレないこと

### 厳格ルール
- 数値でのサイズ指定（pt, px）をプロンプトに含めない
- フォント名をプロンプトに含めない
- 「大きい」「中くらい」「小さい」等の相対表現のみ使用
- すべて手書きマーカー風。デジタルフォントは禁止
- **文字は必ず水平に書く。斜めや曲がった配置は禁止**

## ホワイトボード面の表現

- 清潔感のある白い表面
- 微かな光の反射があってもよい（控えめに）
- 過去のマーカー跡（dry-erase residue）は極めて薄く
- **全スライドで同じ質感・明るさを維持**（一部だけ暗い・明るいは不可）

## レイアウトグリッド（統一性のため）

全スライドで以下のゾーンを統一する：

```
┌─────────────────────────────────────────────┐
│ [ヘッダー: 青・小] ←── 左上固定             │
│                                              │
│ ┌──────────────────────────────────────┐    │
│ │  タイトルゾーン（上部 15%）           │    │
│ │  ページタイトル + アンダーライン       │    │
│ └──────────────────────────────────────┘    │
│                                              │
│ ┌──────────────────────────────────────┐    │
│ │  コンテンツゾーン（中央 70%）         │    │
│ │  テキスト + 図解                      │    │
│ │                                       │    │
│ │                                       │    │
│ └──────────────────────────────────────┘    │
│                                              │
│                            [ページ番号] ──→ │
└─────────────────────────────────────────────┘
```

- ヘッダー位置: 左上角から一定のマージン
- タイトル位置: 上部15%の範囲に収める
- コンテンツ: 中央70%
- ページ番号: 右下角
- **マージンは全スライドで統一**

## ビジュアル要素

### 手描きイラスト
- ホワイトボードに描かれた落書き（doodle）スタイル
- 使用色は黒・青・赤・緑のみ
- シンプルなスケッチ（複雑すぎない）
- **線の太さはテキストの本文と同程度**

### 手描き図解
- フローチャート: 少し歪んだ枠と矢印
- サイクル図: 太い曲線矢印
- 比較表: T字チャート
- すべて黒・青・赤・緑のマーカーのみで描く

### 注釈（アノテーション）
- 重要語を赤い手描き丸で囲む
- 手描きアンダーライン
- 接続矢印
- 星マーク・チェックマーク
