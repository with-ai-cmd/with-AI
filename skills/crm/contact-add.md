# コンタクト登録スキル

## 概要
名刺の写真、音声メモ、テキスト情報などからコンタクト情報を解析し、Notion人脈DBに自動登録する。

## 環境変数の読み込み
以下のファイルから環境変数を読み込んでください:
- `{{WITHAI_ROOT}}/skills/documents/クロードコード/.env`

## 自動実行モード（毎朝8:00）
スケジュール実行時は以下のフローで完全自動処理する:
1. 名刺フォルダ内の未処理画像を全てスキャン
2. 各画像をOCR解析
3. デフォルト値で自動登録:
   - ステータス: 初回接触
   - 業界: 名刺から推定（不明なら「その他」）
   - 出会った場所: 空欄
   - サービス関心: 空欄
   - メモ: 空欄
4. 処理済みフォルダに移動
5. 処理結果サマリーを出力（登録件数・スキップ件数）
6. 画像がなければ「未処理の名刺なし」で終了

## 名刺フォルダの定義
- **名刺フォルダ:** `~/Library/CloudStorage/GoogleDrive-reis.kaito06112323@gmail.com/マイドライブ/名刺/`
- **処理済みフォルダ:** `~/Library/CloudStorage/GoogleDrive-reis.kaito06112323@gmail.com/マイドライブ/名刺/処理済み/`

## 入力の判定
ユーザーからの入力を判定し、以下のいずれかの処理を行う:

### パターン0: 入力なし（`/contact-add` のみ実行された場合）
1. 名刺フォルダ内の画像ファイル（jpg, jpeg, png, heic）を検索する（処理済みフォルダは除外）
2. 画像が見つかった場合:
   - 複数ある場合は一覧を表示し、どれを処理するか確認する（「すべて」も選択可能）
   - 1枚の場合はそのまま解析に進む
3. 画像が見つからない場合:
   - 「名刺フォルダに画像がありません。Google Driveの「名刺」フォルダに写真をアップしてください。」と案内する

### パターン1: 名刺画像パスが提供された場合
1. 画像をReadツールで読み込む
2. 画像から以下の情報をOCR・解析する:
   - 氏名、会社名、役職、メール、電話番号、住所、業界（推定）
3. 解析結果をユーザーに確認してもらう

### パターン2: 音声ファイルが提供された場合
1. Whisper APIまたは利用可能な文字起こしツールで文字起こし
2. 文字起こし結果からコンタクト情報を抽出・構造化
3. 解析結果をユーザーに確認してもらう

### パターン3: テキスト情報が提供された場合
1. テキストからコンタクト情報を抽出・構造化
2. 解析結果をユーザーに確認してもらう

## ユーザーへの追加質問（手動実行時のみ）
解析結果に加えて、以下を確認する:
- 出会った場所（展示会、紹介、SNS等）
- サービス関心（AIKOMON、AI相談、システム開発、AI研修、採用支援）
- 業界（名刺から推定した値を提示し、修正があれば受け付ける）
  - 選択肢: IT・テクノロジー / 不動産 / 士業・法律 / 飲食 / 美容 / 医療・福祉 / 教育 / 金融・保険 / 製造 / 小売・EC / 広告・メディア / 建設 / 人材 / その他
- メモ（人柄、話した内容など自由記述）

## Notion登録
確認が取れたら、以下のAPIコールでNotionのコンタクトDBに登録する:

```bash
curl -s -X POST "https://api.notion.com/v1/pages" \
  -H "Authorization: Bearer $NOTION_API_TOKEN" \
  -H "Notion-Version: 2022-06-28" \
  -H "Content-Type: application/json" \
  -d '{
    "parent": {"database_id": "$NOTION_CONTACT_DB"},
    "properties": {
      "氏名": {"title": [{"text": {"content": "【氏名】"}}]},
      "会社名": {"rich_text": [{"text": {"content": "【会社名】"}}]},
      "役職": {"rich_text": [{"text": {"content": "【役職】"}}]},
      "メール": {"email": "【メール】"},
      "電話": {"phone_number": "【電話】"},
      "ステータス": {"select": {"name": "初回接触"}},
      "業界": {"select": {"name": "【業界】"}},
      "出会った場所": {"rich_text": [{"text": {"content": "【場所】"}}]},
      "初回接触日": {"date": {"start": "【今日の日付 YYYY-MM-DD】"}},
      "サービス関心": {"multi_select": [{"name": "【関心サービス】"}]},
      "メモ": {"rich_text": [{"text": {"content": "【メモ】"}}]}
    }
  }'
```

## 登録完了後
- 登録結果を簡潔に表示
- NotionのページURLを表示して確認を促す
- **処理した名刺画像を「処理済み」フォルダに移動する**
