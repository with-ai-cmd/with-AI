# HP/LP制作 ヒアリング〜見積・請求システム

## 概要
Google Form + Google Spreadsheet でHP/LP制作の受注フローを一元管理するシステム。

## ファイル構成
- `01_create_hearing_form.gs` — GAS: フォーム・スプレッドシート自動生成スクリプト

## セットアップ手順

### 1. Google Apps Scriptプロジェクト作成
1. [script.google.com](https://script.google.com) にアクセス
2. 「新しいプロジェクト」をクリック
3. `01_create_hearing_form.gs` のコードを貼り付け
4. プロジェクト名を「HP制作ヒアリングシステム」に変更

### 2. スクリプト実行
1. 関数セレクタで `createHearingSystem` を選択
2. 「実行」ボタンをクリック
3. 初回はGoogleアカウントの権限承認が必要（「詳細」→「安全でないページに移動」）
4. 完了するとURLがポップアップ表示される

### 3. フォーム回答→見積書自動生成の設定
1. 作成されたスプレッドシートを開く
2. メニュー「拡張機能」→「Apps Script」
3. `onFormSubmit` 関数のコードが入っていることを確認
4. 左メニューの「トリガー」（時計アイコン）をクリック
5. 「トリガーを追加」→以下を設定:
   - 実行する関数: `onFormSubmit`
   - イベントのソース: スプレッドシートから
   - イベントの種類: フォーム送信時
6. 保存

## 自動生成されるもの

### Google Form（ヒアリングシート）
- 基本情報（会社名、担当者、業種）
- 制作目的・ゴール（HP/LP、目的、方向性）
- デザインの好み（雰囲気、色、参考URL）
- ページ構成（必要ページの選択）
- 追加機能（オプション選択）
- 素材について（ロゴ、写真、テキスト）
- 運用方針（更新頻度、保守サポート）
- 予算・スケジュール
- 現在のサイト・ドメイン

### Google Spreadsheet（管理シート）
| シート名 | 用途 |
|----------|------|
| フォームの回答 | フォーム回答が自動で蓄積 |
| 料金表 | ページ単価・追加機能単価の管理 |
| 見積書 | 見積書テンプレート（自動生成対応） |
| 請求書 | 請求書テンプレート |
| コンテンツヒアリング | ページ別の掲載内容ヒアリング |
| 案件管理 | 案件の進捗・金額管理 |

## 運用フロー
1. お客様にフォームURLを送付
2. 回答が届くとスプレッドシートに自動記録
3. 見積書が自動生成される

5. 確定後、請求書シートに転記して発行
