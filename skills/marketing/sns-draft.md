# SNS投稿ドラフト生成スキル

## トリガー

以下のキーワードで起動する:
- 「SNS投稿」「投稿案」「X投稿」「LinkedIn」「sns draft」
- 朝ルーティン（ohayou）の一環としても実行される

## 実行手順

### Step 1: 最新AIニュースの取得

Notion NEWS_RAW データベースから最新のニュースを取得する。

- データベースID: 環境変数 `$NOTION_RAW_DB_ID` を使用
- APIキー: 環境変数 `$NOTION_NEWS_API_KEY` を使用
- Notion APIを叩いて、直近のニュースエントリを取得する（最大10件程度を取得し、その中から選定する）

```bash
curl -s -X POST "https://api.notion.com/v1/databases/${NOTION_RAW_DB_ID}/query" \
  -H "Authorization: Bearer ${NOTION_NEWS_API_KEY}" \
  -H "Notion-Version: 2022-06-28" \
  -H "Content-Type: application/json" \
  -d '{"page_size": 10, "sorts": [{"timestamp": "created_time", "direction": "descending"}]}'
```

### Step 2: ニュースの選定

取得したニュースから **1〜3件** を選ぶ。選定基準:

- **ターゲット読者**: 中小企業の経営者、AI導入を検討している人
- **優先度が高いもの**:
  - 中小企業のAI活用に直結する実務的なニュース
  - AI導入のコスト・効果に関わるもの
  - 業務効率化・生産性向上に繋がるもの
  - 新しいAIツール・サービスのリリース（実用的なもの）
- **避けるもの**:
  - 技術的すぎて経営者に響かないもの
  - 大企業・研究機関向けの話題のみのもの

### Step 3: ドラフト生成

選定した各ニュースについて、以下の2種類のドラフトを生成する。

#### X (Twitter) ドラフト

- **文字数**: 140字以内（厳守）
- **ハッシュタグ**: 2〜3個（例: #AI活用 #中小企業DX #with_AI）
- **トーン**: 端的でインパクトがある。問いかけや気づきを与える形が望ましい
- **視点**: 単なるニュース紹介ではなく、with-AIとしての見解・示唆を必ず入れる

#### LinkedIn ドラフト

- **文字数**: 300〜500字
- **トーン**: ビジネスプロフェッショナル寄りだが、堅すぎない。読みやすさ重視
- **構成**:
  1. ニュースの要点（1〜2文）
  2. with-AI/勝又としての見解・分析（なぜこれが重要か）
  3. 中小企業にとっての実務的な示唆・アクション
  4. 締めの一言（問いかけやCTA）
- **視点**: AI導入の実務的な観点を必ず含める。「で、うちの会社はどうすればいいの？」に答える内容

### トーン & マナー（共通）

- **一人称**: 「勝又」または「with-AI」を使う
- **スタンス**: プロフェッショナルだが親しみやすい
- **核心**: ニュースの横流しではなく、実際のビジネス価値に必ず結びつける
- **避ける表現**: 過度な煽り、根拠のない断言、技術用語の羅列

## 出力フォーマット

選定した各ニュースごとに、以下の形式で出力する:

```
---
## ニュース 1: [ニュースタイトル/要約]

**元記事**: [URLがあれば記載]
**要約**: [2〜3文でニュースの概要]

### X (Twitter) ドラフト

[投稿本文]

(○○字)

### LinkedIn ドラフト

[投稿本文]

(○○字)
---
```

全ニュース分のドラフトを出力した後、ユーザーにレビュー・編集を促す:

> ドラフトを作成しました。修正したい箇所や、トーンの調整があればお知らせください。OKであればそのまま投稿準備に進みます。
