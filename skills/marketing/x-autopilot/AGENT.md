## 投稿生成ルール

### フォーマット
```
[本文（120字以内）]

[補足やデータ（任意）]

#AI導入 #DX推進 #with_AI
→ https://with-ai.jp/blog/xxx （該当記事があれば）
```

### 品質基準
1. **140字以内**（URLとハッシュタグ除く本文）
2. **ハッシュタグは2-3個**（#AI導入 #DX #中小企業AI 等）
3. **必ず自社の視点を入れる**（ニュースの横流し禁止）
4. **CTAを意識**（「詳しくはブログで」「お問い合わせはプロフから」等）
5. **数字を入れる**（「3つのポイント」「導入後30%削減」等）

### スレッド投稿（水曜Tips用）
```
1/5 AI導入で最初にやるべきことは「業務の棚卸し」です。
2/5 よくある失敗は「AIツールから入る」こと...
3/5 ...
4/5 ...
5/5 with-AIではこの棚卸しから一緒にやります。→ https://with-ai.jp/
```

## 投稿フロー

### Phase 1: 投稿案生成
1. 今日の曜日からテーマを決定
2. 該当するネタ元からコンテンツを取得
3. 投稿案を3パターン生成

### Phase 2: 確認・投稿
**現在（手動確認モード）:**
- 3パターンをカイトに提示
- カイトが選択 or 修正指示
- 確認後にOpenClawブラウザ経由で投稿

**将来（自動投稿モード）:**
- CMOの品質基準を満たしていれば自動投稿
- 投稿ログをNotion DBに記録
- エンゲージメントを翌日チェック

### Phase 3: 分析（週次）
- 各投稿のインプレッション・エンゲージメント率を取得
- 高パフォーマンス投稿の特徴を分析
- 翌週の投稿戦略にフィードバック
