# 請求書一斉発行スキル

## 概要
NotionクライアントDBから「契約中」の全クライアントを取得し、
請求書PDFを一括生成 → Gmail下書きを一斉作成する。

## 環境変数の読み込み
- `{{WITHAI_ROOT}}/skills/documents/クロードコード/.env`

## 設定ファイルの読み込み
- `{{WITHAI_ROOT}}/company/invoices/config.json`

---

## 手順

### 1. 請求対象月の確認

```
請求書一斉発行を開始します。
対象月: 2026年4月（デフォルト: 今月）
変更しますか？
```

### 2. 契約中クライアントの一覧取得

Notion MCP toolでクライアントDB（ID: `611e82b2-a57e-4471-b62d-28dcb1d8ffa4`）から
「契約状況 = 契約中」のクライアントを全件取得する。

**取得プロパティ:**
- 会社名・顧客名
- 担当者名
- 契約プラン
- 月額
- 契約開始日
- 請求先メール（なければメールアドレス）
- 請求サイクル
- 住所

### 3. 各クライアントの金額計算

クライアントごとに以下を計算:

#### 日割り判定
```python
if 契約開始日の年月 == 請求対象月:
    # 日割り計算
    remaining_days = 月の日数 - 契約開始日の日 + 1
    amount = int(月額 * remaining_days / 月の日数)
else:
    # 通常月
    amount = 月額
```

#### スポット案件のスキップ
月額が0またはNullのクライアントはスキップし、一覧に表示する。

### 4. 請求一覧の表示と確認

```

----|---------------------|--------------|------------|-----------|--------

 3  | 株式会社CCC          | AIKOMONプラン  | ¥133,333   | ¥146,666  | 日割り(4/15〜)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

この内容で一斉発行しますか？（個別に修正がある場合は番号を指定してください）
```

### 5. 請求書PDF一括生成

確認後、各クライアントのPDFを順次生成する:

```bash
# クライアントごとにディレクトリ確認・作成
mkdir -p {{WITHAI_ROOT}}/clients/{会社名}/{contracts,invoices,notes}

# PDF生成
python3 {{WITHAI_ROOT}}/company/invoices/scripts/generate_invoice.py \
  --company "{会社名}" \
  --contact "{担当者名}" \
  --item "{プラン名}（{対象月}分）" \
  --amount {金額} \
  --tax {消費税} \
  --invoice-number "{請求書番号}" \
  --invoice-date "{請求日}" \
  --due-date "{支払期限}" \
  --output "{{WITHAI_ROOT}}/clients/{会社名}/invoices/{YYYY-MM}_請求書.pdf"
```

### 6. Gmail下書き一斉作成

各クライアントに対してGmail MCP toolで下書きを作成する。

**メールテンプレート:**

```
宛先: {請求先メール}
件名: 【with-AI株式会社】{対象月}分 ご請求書送付のご案内

本文:
{担当者名} 様

いつもお世話になっております。
with-AI株式会社の勝又です。

{対象月}分のご請求書を添付にてお送りいたします。

━━━━━━━━━━━━━━━━━━
請求書番号: {請求書番号}
ご請求金額: ¥{税込合計}（税込）

━━━━━━━━━━━━━━━━━━

お振込先は請求書に記載しております。
ご確認のほど、よろしくお願いいたします。

ご不明な点がございましたら、お気軽にお問い合わせください。

--
with-AI株式会社
代表取締役 勝又海斗
Email: info@with-ai.jp
Tel: 090-1296-4814
Web: https://with-ai.jp/
```

**Gmail下書き作成:**
`gmail_create_draft` ツールを使用する。

> 注意: PDFの添付はGmail MCP toolの制約により手動で行う必要がある場合があります。
> その場合は下書き作成後に「以下のPDFを各下書きに添付してください」と案内します。
