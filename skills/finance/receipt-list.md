# 領収書一覧スキル

## 概要
Notion領収書DBから領収書を検索・一覧表示する。

## 環境変数の読み込み
```bash
source ~/Desktop/クロードコード/.env
```

## 引数の解析
$ARGUMENTSを解析し、フィルタ条件を構築する。

| 引数例 | フィルタ |
|-------|---------|
| 引数なし | 今月の領収書 |
| `3月` or `2026-03` | 指定月 |
| `Amazon` | 支払先に含む |
| `ソフトウェア` | カテゴリ一致 |
| `未確認` | ステータス一致 |
| `今年` or `2026年` | 年間 |
| `Q1` | 1-3月 |

複数条件の組み合わせも可能（例: `3月 ソフトウェア`）

## 検索実行

### デフォルト（今月）
```bash
curl -s -X POST "https://api.notion.com/v1/databases/$NOTION_RECEIPT_DB/query" \
  -H "Authorization: Bearer $NOTION_API_TOKEN" \
  -H "Notion-Version: 2022-06-28" \
  -H "Content-Type: application/json" \
  -d '{
    "filter": {
      "and": [
        {"property": "日付", "date": {"on_or_after": "【月初 YYYY-MM-01】"}},
        {"property": "日付", "date": {"on_or_before": "【月末 YYYY-MM-DD】"}}
      ]
    },
    "sorts": [{"property": "日付", "direction": "descending"}],
    "page_size": 100
  }'
```

## 注意事項
- NOTION_RECEIPT_DB が .env に存在しない場合は案内を表示
- 100件以上ある場合はページネーション対応
