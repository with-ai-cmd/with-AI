# 領収書集計スキル

## 概要
Notion領収書DBのデータを集計し、カテゴリ別・月別のレポートを表示する。

## 環境変数の読み込み
```bash
source ~/Desktop/クロードコード/.env
```

## 引数の解析

| 引数例 | 集計範囲 |
|-------|---------|
| 引数なし | 今月のカテゴリ別集計 |
| `3月` | 指定月のカテゴリ別集計 |
| `2026年` or `今年` | 年間の月別×カテゴリ別集計 |
| `Q1` | 四半期集計（1-3月） |
| `確定申告` | 前年1-12月の年間集計（確定申告用） |

## データ取得
指定期間のすべての領収書を取得する。

```bash
curl -s -X POST "https://api.notion.com/v1/databases/$NOTION_RECEIPT_DB/query" \
  -H "Authorization: Bearer $NOTION_API_TOKEN" \
  -H "Notion-Version: 2022-06-28" \
  -H "Content-Type: application/json" \
  -d '{
    "filter": {
      "and": [
        {"property": "日付", "date": {"on_or_after": "【期間開始】"}},
        {"property": "日付", "date": {"on_or_before": "【期間終了】"}}
      ]
    },
    "sorts": [{"property": "日付", "direction": "ascending"}],
    "page_size": 100
  }'
```

100件以上ある場合は `has_more` / `next_cursor` で追加取得する。

### 確定申告用
年間集計に加えて:
- 未確認ステータスの領収書がある場合は警告
- 経費率の高いカテゴリのハイライト

## 注意事項
- NOTION_RECEIPT_DB が .env に存在しない場合は案内を表示
- 金額のフォーマットは3桁カンマ区切り、¥マーク付き
- パーセンテージは小数点1桁
