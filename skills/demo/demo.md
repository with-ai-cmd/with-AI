# /demo — 商談デモ: ワンショット並列生成

## 概要
商談相手の情報をその場で聞いて、5つのビジネス文書を同時生成するデモ。
「今お聞きした情報で、もう全部できちゃいました」を実現する。

## ヒアリング

ユーザーの指示から以下を読み取る。不足しているものだけ聞く:

1. **会社名** — 相手の会社名（例: 株式会社タイム）
2. **担当者名** — 商談相手の名前（例: 田中太郎）
3. **役職** — 相手の役職（例: 営業部長）※ 不明なら省略OK
4. **メールアドレス** — 相手のメール ※ 不明なら省略OK
5. **提案内容** — 今回提案しているサービス名（例: AIKOMONプラン、AI研修、システム開発 など自由記述）
6. **金額** — 提案している月額 or 総額（税抜）

聞き方の例:
```
了解！デモ作るね。ちょっと教えて:
- 会社名と担当者の名前は？
- 今回どんな内容を提案してる？
- 金額はいくらくらい？
```

すでに指示の中に情報が含まれている場合は、確認だけして即実行する:
```
こんな感じでいい？
  会社: 株式会社タイム
  担当: 田中太郎さん
  提案: AIKOMONプラン — ¥250,000/月
OKならすぐ作るよ！
```

## 実行

```bash
python3 {{WITHAI_ROOT}}/skills/demo/demo-generate.py \
  --company "{会社名}" \
  --contact "{担当者名}" \
  --title "{役職}" \
  --email "{メールアドレス}" \
  --plan "{提案内容}" \
  --amount {金額（税抜・数値）} \
  --output-dir ~/Desktop/demo-{会社名の短縮形}
```

※ 役職・メールが不明の場合は --title "" --email "" で空文字を渡す
