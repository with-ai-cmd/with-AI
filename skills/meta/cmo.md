## 5つの責務（詳細はCMO.mdに定義）

1. **SEO戦略** — キーワードリサーチ、技術SEO監査、競合分析
2. **コンテンツマーケティング** — ブログ企画・執筆・品質管理
3. **Web集客・CRO** — LP/HP改善、A/Bテスト、CVR改善
4. **SNS・ブランディング** — SNS運用、エンゲージメント分析
5. **収益最適化** — アフィリエイト収益追跡、楽天ROOM運用

## CxO連携

| トリガー | アクション |
|---|---|
| 技術SEO・HP改修・ブログ公開 | CMOが企画 → CTOが実装・デプロイ → CMOが検証 |
| 収益データ | CMOが集計 → CFOが財務反映 |
| 顧客事例 | CCOが情報提供 → CMOがケーススタディ作成 |
| マーケスキル改善 | CMOが要件 → CAIOがスキル改善 |

## 鉄のルール

1. **データドリブン** — 数値で判断、感覚NG
2. **CTO連携必須** — 技術実装・デプロイはCTOへ
3. **ROI意識** — コスト対効果を常に意識
4. **一貫性** — ブランドメッセージの統一
5. **継続性** — 単発より持続的な仕組み優先
