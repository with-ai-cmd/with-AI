# デイリータスク確認スキル

## 概要
今日が期日のタスクを一覧表示し、完了/未完了を確認する。
未完了のタスクは空いている日に自動でリスケジュールする。

## 環境変数の読み込み
以下のファイルから環境変数を読み込んでください:
- `~/Desktop/クロードコード/.env`

## ステップ1: 今日のタスクを取得
今日の日付（YYYY-MM-DD）を基準に、期日が今日以前かつ未完了のタスクを取得する。

```bash
curl -s -X POST "https://api.notion.com/v1/databases/$NOTION_TASK_DB/query" \
  -H "Authorization: Bearer $NOTION_API_TOKEN" \
  -H "Notion-Version: 2022-06-28" \
  -H "Content-Type: application/json" \
  -d '{
    "filter": {
      "and": [
        {
          "property": "期日",
          "date": {"on_or_before": "【今日の日付 YYYY-MM-DD】"}
        },
        {
          "property": "ステータス",
          "select": {"does_not_equal": "完了"}
        }
      ]
    },
    "sorts": [{"property": "期日", "direction": "ascending"}]
  }'
```

## ステップ2: タスク一覧を表示
取得したタスクを以下の形式で表示する:

```
📋 今日までのタスク（YYYY-MM-DD）

| # | タスク名 | 期日 | 優先度 | 関連コンタクト |
|---|---------|------|--------|--------------|
| 1 | ○○○   | 3/18 | 高     | △△さん       |
| 2 | ×××   | 3/17 | 中     | □□さん       |（※期限超過）

完了したタスクの番号を教えてください（例: 1,3 または「全部」または「なし」）
```

## ステップ3: 完了処理
ユーザーが指定したタスクのステータスを「完了」に更新する。

```bash
curl -s -X PATCH "https://api.notion.com/v1/pages/【ページID】" \
  -H "Authorization: Bearer $NOTION_API_TOKEN" \
  -H "Notion-Version: 2022-06-28" \
  -H "Content-Type: application/json" \
  -d '{
    "properties": {
      "ステータス": {"select": {"name": "完了"}}
    }
  }'
```

## ステップ4: 未完了タスクのリスケジュール
未完了のタスクについて:

1. 今後1週間のタスク状況を取得して、各日の負荷を把握する

```bash
curl -s -X POST "https://api.notion.com/v1/databases/$NOTION_TASK_DB/query" \
  -H "Authorization: Bearer $NOTION_API_TOKEN" \
  -H "Notion-Version: 2022-06-28" \
  -H "Content-Type: application/json" \
  -d '{
    "filter": {
      "and": [
        {"property": "期日", "date": {"after": "【今日の日付】"}},
        {"property": "期日", "date": {"on_or_before": "【1週間後の日付】"}},
        {"property": "ステータス", "select": {"does_not_equal": "完了"}}
      ]
    }
  }'
```

2. 空いている日（タスクが少ない日）に自動でリスケする
   - 土日は避ける（営業日のみ）
   - 1日あたりのタスクが偏らないよう分散させる
   - 優先度「高」のタスクは直近の空き日に入れる
   - 優先度「中・低」は余裕のある日に入れる

3. リスケ案をユーザーに提示する:

```
🔄 リスケ案

| タスク名 | 元の期日 | → 新しい期日 | 理由 |
|---------|---------|-------------|------|
| ○○○   | 3/18   | → 3/19      | 翌営業日（空き） |
| ×××   | 3/17   | → 3/20      | 高優先度だが3/19は詰まっているため |

この内容でリスケしますか？（はい / 調整したい番号を指定）
```

4. 確認後、期日を更新しステータスを「リスケ」に変更する

```bash
curl -s -X PATCH "https://api.notion.com/v1/pages/【ページID】" \
  -H "Authorization: Bearer $NOTION_API_TOKEN" \
  -H "Notion-Version: 2022-06-28" \
  -H "Content-Type: application/json" \
  -d '{
    "properties": {
      "期日": {"date": {"start": "【新しい期日 YYYY-MM-DD】"}},
      "ステータス": {"select": {"name": "リスケ"}}
    }
  }'
```

## ステップ5: サマリー表示
最後に今日の結果をまとめて表示する:

```
✅ 完了: X件
🔄 リスケ: Y件
📅 明日のタスク: Z件（一覧表示）
```
