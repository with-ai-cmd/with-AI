# 学習ノート生成スキル

## トリガー
`/learning-note` または「学習ノート作成」「今日の学習まとめ」

## 処理フロー

1. `.env` から環境変数を読み込む
2. Notion「学習インボックス」DBから未処理のURLを全件取得
3. 各URLのコンテンツを WebFetch で取得
4. 取得した記事群をまとめて要約・構造化:
   - 記事ごとの要点（3行以内）
   - なぜ重要か（チームにとっての意味）
   - カテゴリ分け（AI / ビジネス / 技術 / その他）
   - 全体を通じた学びまとめ
5. Notion「学習ノート」DBに日次記事を作成
6. インボックスのステータスを「処理済み」に更新
7. 完了レポートを表示

## 完了レポート
```
========================================
 学習ノート生成 完了レポート
========================================
未処理記事:    XX件
学習ノート:    [Notion URL]
カテゴリ:      AI, 技術
========================================
```
